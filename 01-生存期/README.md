## 💡 生存期生存法则

### ✅ 应该做的
- 优先学习**基础保护**，确保不被坑
- 建立**每日循环**，养成工作习惯
- 每月做**状态检查**，了解自己的位置
- 积极参与**社区共建**，向他人学习
- 接受**合理的低价**（但不要贱卖）

### ❌ 不应该做的
- 不要接**没有合同**的项目
- 不要接受**免费试稿**（除非有明确的有偿转化路径）
- 不要**过度承诺**（说你能做到你做不到的事）
- 不要**忽视健康**（熬夜、不运动、不吃饭）
- 不要**孤立自己**（加入社区，寻求帮助）

## 🤖 AI搭档策略（生存期）

在生存期，把AI当作**导师**——一个经验丰富、随时在线的导师。

### 生存期AI使用场景

**1. 合同审查**
```
"我收到了一份合同，帮我看看有没有陷阱。"
"这份合同的付款条款合理吗？"
```

**2. 报价建议**
```
"客户问我预算，我不知道怎么报。"
"这个项目我应该报多少？考虑我的经验和市场行情。"
```

**3. 客户沟通**
```
"客户说'预算有限'，我该怎么回复？"
"帮我写一封邮件，委婉地拒绝免费试稿。"
```

**4. 技能学习**
```
"我想学习品牌设计，给我制定一个3个月学习计划。"
"用简单的话解释什么是品牌定位。"
```
