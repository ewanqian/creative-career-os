# 沟通指南：应对免费试稿

**保护你的创意价值，同时维持良好的客户关系。**

---

## 为什么这很重要

"先做一稿看看"是创意行业最常见的陷阱之一。

如果你同意免费试稿，可能会面临：
- 投入大量时间和精力，却得不到任何回报
- 创意被"借鉴"，即使没有中标
- 陷入恶性循环，不断被要求修改
- 专业价值被贬低，难以建立合理定价

通过专业的应对策略，你可以：
- 委婉但坚定地拒绝免费试稿
- 提供有偿试稿等替代方案
- 筛选出真正尊重你价值的客户
- 建立健康的合作基础

---

## 三种应对策略

### 策略 A：委婉拒绝法

**适用场景**：客户说"先做一稿看看，合适再付费"

#### 温和但坚定的回应

```
感谢您的认可。关于"先试稿"，我想和您分享一下我的考虑：

设计是一个完整的创作过程，从需求分析到概念构思再到执行，需要投入大量时间和精力。如果先做方案，可能无法充分了解您的需求，反而影响最终效果。

我建议的流程是：
1. 先充分沟通需求，确保我理解您的期望
2. 签订合同并支付预付款后启动项目
3. 提供2-3个不同方向的初稿供选择
4. 根据反馈深化设计

这样既保证质量，也体现双方的诚意。您看是否可以这样安排？
```

#### 专业且有理有据的回应

```
理解您希望看到方案再决定的考虑。不过从专业角度，"先试稿"存在几个问题：

1. **需求不清**
   没有充分调研就出手，方案可能偏离您的真实需求

2. **时间成本**
   每个方案都需要2-3天的深度工作，免费试稿对创作者不公平

3. **版权风险**
   方案一旦流出，即使未被采用，创意也可能被借鉴

我建议的替代方案：

**方案A**：签订合作协议，支付30%预付款，我提供2-3个概念方向

**方案B**：如确实需要先看方案，可以支付[X]元作为概念设计费，该费用可抵扣总项目款

**方案C**：我可以提供过往类似项目的案例和思路，帮助您了解我的设计风格

您倾向于哪种方式？
```

---

### 策略 B：有偿试稿法

**适用场景**：客户坚持要先看方案，但你不想完全拒绝

**你的提议**：
```
理解您的谨慎。为了平衡双方的需求，我建议采用"有偿试稿"的方式：

**试稿方案**：
- 费用：[X]元（可抵扣总项目款）
- 交付：1个概念方向 + 设计思路说明
- 版权：试稿阶段作品著作权归我，您获得审阅权
- 后续：如继续合作，该费用计入总款；如不合作，您不得使用试稿内容

这样既能让您看到我的设计能力，也保护了我的劳动成果。

您觉得这个方案可以接受吗？
```

**关键点**：
- 费用要合理（通常是总项目费的10-20%）
- 明确试稿的交付内容和版权归属
- 强调费用可抵扣，降低客户心理门槛

---

### 策略 C：比稿保护法

**适用场景**：客户明确表示要多家比稿

**你的回应**：
```
了解到贵司正在比稿。参与比稿前，我需要确认几个事项：

**比稿保护条款**：
1. **比稿费用**
   参与比稿需要支付[X]元比稿费（中标后可抵扣，未中标不退还）

2. **方案数量**
   提供[X]个概念方案

3. **版权保护**
   未中标方案的著作权完全归我，贵司不得使用或借鉴

4. **保密协议**
   双方签署保密协议，确保方案不被泄露给第三方

5. **反馈机制**
   无论是否中标，希望获得评委的专业反馈

如果以上条款可以接受，我很乐意参与比稿。
```

**关键点**：
- 比稿费可以筛选掉不认真的客户
- 版权保护防止创意被"白嫖"
- 保密协议保护双方利益

## 识别"白嫖"的早期信号

以下信号表明客户可能想"白嫖"创意，需要提高警惕：

- ⚠️ 急于看方案，但对合同和付款含糊其辞
- ⚠️ 要求"先做一稿看看"，不提费用
- ⚠️ 比稿没有明确的评选标准和费用
- ⚠️ 客户内部"决策人"众多，意见难以统一
- ⚠️ 对设计流程和专业性缺乏基本尊重

**发现这些信号时，坚决拒绝免费试稿。**

## 常见情况应对

### 情况 1：客户说"就试一稿，不行就算了"

**你的回应**：
```
理解您的想法。不过即使是一稿，也需要：

1. 需求调研和分析（半天）
2. 概念构思和草图（1-2天）
3. 设计执行和完善（1-2天）

总计需要2-3天的专业工作。

作为专业人士，我需要对时间和质量负责。如果免费试稿，我无法保证投入足够的精力。

如果您确实需要先看方案，我建议：
- 支付[X]元概念设计费，我提供1个完整方案
- 或者先签订合同，支付30%预付款，我提供2-3个方向

这样既能让您看到效果，也确保我能全力以赴。
```

### 情况 2：客户说"别的设计师都愿意免费试"

**你的回应**：
```
理解，市场上确实有不同的做法。

每个设计师有自己的工作方式和价值标准。我的做法是：

1. **充分沟通**：先深入了解需求，确保方向正确
2. **专业承诺**：签订合同后全力以赴，提供高质量方案
3. **版权保护**：保护双方权益，避免后续纠纷

如果您希望先看到方案再决定，我可以：
- 提供过往类似项目的详细案例
- 安排一次深入的需求沟通会议
- 提供概念设计服务，费用[X]元（可抵扣总款）

您觉得哪种方式更合适？
```

### 情况 3：客户是朋友介绍，不好意思拒绝

**你的回应**：
```
感谢[朋友名字]的推荐，我也很重视这次合作。

正因为是朋友介绍，我更要对双方负责：

1. **对朋友负责**：确保项目顺利完成，不让朋友为难
2. **对您负责**：提供专业服务，确保最终效果
3. **对自己负责**：合理安排时间和精力

所以我建议按标准流程进行：
- 充分沟通需求
- 签订合同，支付30%预付款
- 提供2-3个概念方向
- 根据反馈深化设计

这样既保证质量，也对朋友有个交代。您看可以吗？
```

## 核心原则

### ✅ 要做的

1. **态度温和但立场坚定**
   - 拒绝时要保持专业和礼貌，不要指责客户

2. **提供替代方案**
   - 不要只说"不行"，要给出可行的替代建议

3. **强调专业价值**
   - 让客户理解设计工作的专业性和价值

4. **准备退出**
   - 如果客户坚持免费试稿，要有勇气放弃

5. **保留证据**
   - 所有沟通记录要保存，以备纠纷

### ❌ 不要做

1. **不要心软妥协**
   - 一旦同意免费试稿，就可能陷入被动

2. **不要贬低自己**
   - 不要说"我水平不够，需要试稿证明"

3. **不要情绪化**
   - 避免"你们就是想白嫖"这样的表达

4. **不要模糊承诺**
   - 不要说"我先试试"，要说"按流程来"
