# 合同指南：付款节点设置

**通过分阶段付款，确保你的劳动成果得到及时回报。**

---

## 为什么这很重要

创意项目的最大风险之一，就是"做完项目收不到钱"。

通过合理设置付款节点，你可以：
- 降低回款风险，避免项目做完后长期拖欠
- 确保每个阶段的工作都能得到相应报酬
- 保持健康的现金流，支持持续创作
- 建立专业的合作框架，提升客户信任

---

## 三种常见方案

### 方案 A：3331付款模式（推荐）

**核心思路**：将项目分为四个阶段，按3:3:3:1的比例付款，尾款作为质保金。

**适用场景**：
- 金额较大的项目（建议超过1万元）
- 周期较长的项目（超过2周）
- 新客户或需要建立信任的合作

**参考条款**：
```
第X条 费用及支付方式

1. 项目总费用：人民币[X]元（大写：[X]元整）

2. 付款方式：分阶段支付

   第一阶段：合同签订后3个工作日内
   - 支付比例：30%
   - 支付金额：[X]元
   - 对应成果：项目启动、需求确认

   第二阶段：概念方案确认后3个工作日内
   - 支付比例：30%
   - 支付金额：[X]元
   - 对应成果：完成并提交概念设计方案

   第三阶段：深化设计确认后3个工作日内
   - 支付比例：30%
   - 支付金额：[X]元
   - 对应成果：完成并提交深化设计方案

   第四阶段：项目验收合格后7个工作日内
   - 支付比例：10%
   - 支付金额：[X]元
   - 对应成果：完成全部交付物，项目验收合格

3. 付款方式：银行转账
   - 户名：[你的账户名]
   - 开户行：[开户银行]
   - 账号：[银行账号]

4. 发票开具：收到每笔款项后[X]个工作日内开具等额发票

5. 逾期付款：
   - 逾期每日按应付款项的万分之五支付违约金
   - 逾期超过15日，有权暂停工作
   - 逾期超过30日，有权解除合同并要求支付已完成工作的费用
```

**你的收益**：
- 项目开始前就收到30%，覆盖启动成本
- 每个阶段完成后及时回款，保持现金流
- 尾款作为质保金，确保客户配合验收

---

### 方案 B：预付款+验收付款模式

**核心思路**：适用于周期较短或金额较小的项目，简化付款流程。

**适用场景**：
- 金额较小的项目（1万元以内）
- 周期较短的项目（1-2周）
- 有一定信任基础的客户

**参考条款**：
```
第X条 费用及支付方式

1. 项目总费用：人民币[X]元（大写：[X]元整）

2. 付款方式：

   预付款：合同签订后3个工作日内支付50%，即[X]元
   - 项目启动的必要条件
   - 收到预付款后开始计算项目周期

   验收款：项目验收合格后5个工作日内支付50%，即[X]元
   - 确认终稿并签署验收单后支付
   - 收到尾款后交付全部源文件

3. 验收标准：
   - 提交的设计方案符合合同约定的需求
   - 收到终稿后[X]个工作日内完成验收
   - 超过[X]个工作日未提出异议的，视为验收合格

4. 修改与验收：
   - 合同约定的修改次数内，免费修改
   - 超出约定修改次数的，需支付额外费用
   - 因客户原因导致项目延期的，付款时间相应顺延
```

---

### 方案 C：月结模式（长期合作）

**核心思路**：适用于长期合作或按月服务的项目。

**适用场景**：
- 长期合作客户（合作3个月以上）
- 按月服务的项目（如月度设计服务）
- 已建立良好信任关系的客户

**参考条款**：
```
第X条 费用及支付方式（月结模式）

1. 服务费用：
   - 月度服务费：人民币[X]元/月
   - 或按项目计费，单价为[X]元/项目

2. 结算周期：
   - 每月[X]日统计上月工作量
   - 每月[X]日前提交对账单和发票
   - 收到发票后[X]个工作日内付款

3. 预付款（可选）：
   - 合作开始前支付首月服务费作为预付款
   - 或支付[X]元作为合作保证金，合作结束后退还

4. 付款保障：
   - 连续两个月未按时付款的，有权暂停服务
   - 累计欠款超过[X]元的，有权解除合同
```

## 如何选择适合你的方案

**问自己三个问题**：

1. **项目金额多大？**
   - 超过1万元 → 方案A（3331模式）
   - 1万元以内 → 方案B（预付+验收）

2. **客户关系如何？**
   - 新客户 → 方案A或B
   - 长期合作 → 方案C（月结）

3. **项目周期多长？**
   - 超过2周 → 方案A
   - 1-2周 → 方案B
   - 长期服务 → 方案C

## 与客户沟通的话术

### 场景：客户要求"验收后一次性付款"

**温和但坚定的回应**：
```
感谢您的信任。关于付款方式，我建议采用分阶段付款的模式。

这样对我们双方都更有保障：
- 您可以分阶段确认成果，降低风险
- 我可以更专注地投入项目，确保质量

我建议的付款安排是：
- 签约时支付30%启动项目
- 概念方案确认后支付30%
- 深化设计确认后支付30%
- 验收后支付尾款10%

这样既符合行业惯例，也能确保项目顺利推进。您看这样的安排是否可以？
```

**专业且有理有据的回应**：
```
理解贵司的财务流程。从项目管理角度，我建议采用里程碑付款的方式。

原因如下：
1. 风险控制：分阶段确认成果，降低双方风险
2. 现金流：我需要支付团队成本和外包费用，预付款是必要的
3. 项目保障：确保我能全身心投入，不被资金问题分心

我建议的方案：
- 预付款30%：项目启动
- 中期款40%：主要成果交付
- 尾款30%：验收完成

或者，如果贵司确实需要验收后付款，我需要：
1. 提高报价10-15%以覆盖资金成本
2. 签订正式合同并加盖公章
3. 验收期限明确为[X]个工作日

您看哪种方案更合适？
```

### 场景：客户拖延付款

**礼貌但明确的催款**：
```
您好，根据合同约定，[阶段名称]的付款已于[日期]到期，目前还未收到款项。

为确保项目顺利推进，请您尽快安排付款。

付款信息：
- 金额：[X]元
- 账户：[账户信息]

收到款项后，我将立即继续[下一步工作]。

如有特殊情况，请提前告知，我们可以协商解决。
```

## 常见问题

**Q：客户说"我们公司的财务流程是验收后付款"，怎么办？**

A：理解客户的财务流程，但也要坚持自己的原则。可以提出折中方案：
- 接受验收后付款，但要求提高报价10-15%
- 或要求签订正式合同并加盖公章作为保障
- 或要求缩短验收期限（如3-5个工作日）

**Q：已经签了"验收后付款"的合同，还能补救吗？**

A：可以在项目执行过程中，与客户协商变更付款方式。比如：
- "为了确保项目顺利推进，能否先支付30%预付款？剩余70%验收后支付。"
- 或要求客户签署验收单时同时安排付款

**Q：客户拖欠付款，但项目还在进行中，怎么办？**

A：根据合同条款，逾期超过15日有权暂停工作。可以：
- 先礼貌提醒
- 如仍未付款，正式通知暂停工作
- 保留已完成的工作成果，待付款后再继续
