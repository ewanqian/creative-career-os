# 合同指南：著作权归属界定

**在合作开始前，明确创作成果的归属。**

---

## 为什么这很重要

当你为客户创作时，著作权归谁所有？这直接影响：

- 你是否可以在作品集展示这个项目
- 未来是否可以基于这个创作衍生新作品
- 客户使用作品的范围和期限
- 如果客户违约，你是否有权停止使用作品

在合作开始前明确这一点，可以避免后续的争议和不必要的损失。

---

## 三种常见方案

### 方案 A：分级授权（推荐）

**核心思路**：你保留著作权，客户获得特定范围的使用权。

**适用场景**：
- 大多数商业设计项目
- 你希望保留作品控制权
- 客户只需要特定用途

**参考条款**：
```
第X条 知识产权归属与授权

1. 作品著作权归属
   本合同项下创作的作品的完整著作权归乙方（创作者）所有。

2. 授权范围与期限
   甲方（客户）获得以下非独占使用权：
   
   a) 基础使用权：
      - 使用范围：[具体使用场景，如品牌宣传、产品包装]
      - 使用期限：自交付之日起[X年/永久]
      - 使用地域：[中国大陆/全球]
   
   b) 扩展使用权（可选，需另行协商费用）：
      - 衍生开发权：基于作品开发新作品的权利
      - 转授权：允许第三方使用的权利
      - 独占使用权：排除乙方自身使用的权利

3. 署名权保留
   乙方保留在作品上署名的权利，甲方在使用作品时应保留乙方署名。

4. 作品交付与确权
   - 初稿阶段：乙方保留全部权利，甲方仅有审阅权
   - 终稿确认：甲方支付尾款后，获得约定的使用权
   - 源文件：源文件所有权归乙方，甲方可获得约定格式的使用文件
```

**你的收益**：
- 保留著作权，可在作品集展示
- 可授权给其他客户（非竞品）
- 可提供后续延展设计服务

---

### 方案 B：阶段确权

**核心思路**：根据项目阶段逐步明确权利归属。

**适用场景**：
- 长期合作项目
- 分阶段付款的项目
- 需要逐步建立信任的合作

**参考条款**：
```
第X条 分阶段知识产权安排

1. 概念阶段（初稿）
   - 著作权：完全归乙方所有
   - 甲方权利：仅可审阅并提出修改意见
   - 如项目终止：甲方不得使用任何初稿内容

2. 执行阶段（修改稿）
   - 著作权：仍归乙方所有
   - 甲方权利：可内部讨论、向上汇报
   - 限制：不得对外展示、不得用于商业用途

3. 完成阶段（终稿）
   - 甲方支付全部费用后，获得【具体约定的使用权】
   - 乙方保留【署名权/作品集使用权/案例展示权】

4. 特别约定
   - 比稿保护：如采用比稿形式，未中标方案著作权完全归乙方
   - 修改限制：超出约定次数的修改，著作权归属需重新协商
```

---

### 方案 C：著作权转让

**核心思路**：客户获得完整著作权，你保留署名权。

**适用场景**：
- 雇佣性质的项目
- 客户支付足额费用（通常是3-5倍）
- 你不需要在作品集展示

**参考条款**：
```
第X条 著作权转让

1. 作品著作权转让
   甲方支付全部费用后，作品完整著作权转让给甲方。

2. 乙方保留权利
   - 署名权：乙方保留在作品上署名的权利
   - 案例展示权：乙方有权在非公开场合展示该作品作为案例

3. 转让限制
   - 甲方不得将作品用于违法用途
   - 甲方不得损害乙方声誉
```

**注意**：这种模式通常需要客户支付更高费用，因为这意味着你放弃了未来的所有权益。

---

## 如何选择适合你的方案

**问自己三个问题**：

1. **这个项目对你的作品集重要吗？**
   - 重要 → 选择方案A或B，保留展示权
   - 不重要 → 可以考虑方案C

2. **客户的使用范围是什么？**
   - 特定用途 → 方案A，授权特定范围
   - 全方位使用 → 方案C，但要求更高费用

3. **你们的合作关系如何？**
   - 新客户 → 方案B，分阶段确权
   - 长期合作 → 方案A，灵活授权

## 与客户沟通的话术

### 场景：客户说"版权全部归我们"

**温和但坚定的回应**：
```
感谢您的信任。关于版权归属，我建议采用分级授权的方式。

这样既能满足贵司的使用需求，也能保护我们作为创作者的基本权益。

具体而言，贵司可以获得在[具体场景]的完整使用权，而我保留署名权和在作品集中展示的权利。

这样既符合行业惯例，也能确保双方的权益都得到保障。您看这样的安排是否可以？
```

**专业且有理有据的回应**：
```
理解贵司希望获得完整使用权的考虑。不过，从专业角度建议采用"授权使用"而非"买断版权"的模式。

原因如下：
1. 成本控制：买断版权通常需要支付3-5倍的费用，而授权使用更经济
2. 灵活性：您可以根据实际需要选择授权范围和期限
3. 后续服务：保留版权有利于我们提供后续的延展设计和更新服务

我建议的授权方案是：贵司获得[具体范围]的永久使用权，费用为[金额]，这样既满足需求，又合理控制成本。
```

**对方坚持时的谈判策略**：
```
如果贵司确实需要买断版权，我需要调整报价。因为这意味着：
1. 我无法再使用该作品作为案例
2. 失去了后续授权给其他客户的潜在收益
3. 需要签署更严格的保密和竞业限制

基于以上考虑，买断版权的报价为[原报价的3-5倍]。

或者，我们可以折中：贵司获得独占使用权（排除我授权给竞品），但保留我的署名权和案例展示权，费用为[原报价的1.5-2倍]。
```

## 常见问题

**Q：客户说"不转让版权就不合作"，怎么办？**

A：评估这个客户对你的重要性。如果是重要客户且费用合理，可以考虑方案C。但如果费用不够高，建议坚持方案A或B。记住，版权是你的核心资产之一。

**Q：已经签了"版权全部归甲方"的合同，还能补救吗？**

A：可以尝试与客户协商补充协议，争取保留署名权和案例展示权。如果客户不同意，至少确保你在非商业场合（如面试、获奖申报）可以展示。

**Q：开源作品、模板作品怎么算？**

A：如果是基于开源作品修改，需要遵循原作品的授权协议。如果是使用模板，确保你有商业使用授权。这些都需要在合同中明确说明。
