## 💡 紧急情况通用原则

### 第一步：稳住情绪（30秒）

**深呼吸**：
1. 吸气4秒 → 屏息4秒 → 呼气6秒
2. 重复3次

**告诉自己**：
- "这不是世界末日"
- "我有能力解决这个问题"
- "最坏的情况是什么？我能承受吗？"

### 第二步：评估严重程度（1分钟）

| 等级 | 特征 | 处理方式 |
|:---:|:---|:---|
| 🔴 **紧急** | 几小时内必须处理，有严重后果 | 立即行动，暂停其他工作 |
| 🟡 **重要** | 24小时内需要处理，有实质影响 | 优先处理，合理安排时间 |
| 🟢 **一般** | 可以协商，影响有限 | 正常处理，不必焦虑 |

### 第三步：选择解决路径

**能自己解决** → 查看下方具体方案

**需要建议** → 使用 [AI应急助手](#ai应急助手)

**需要支持** → 寻求朋友/导师/专业人士帮助

## 📋 具体情况解决方案

### 情况1：deadline 临近，做不完了

**紧急程度**：🔴 紧急

**立即行动**：

**选项A：协商延期（推荐）**
```
[客户称呼]，关于[项目名称]，我需要坦诚沟通。

目前进度是[X]%，还剩[具体工作]未完成。
按原计划明天交付，质量可能无法保证。

建议两个方案：
1. 延期[X天]，保证质量
2. 明天交付初版，[X天后]交付完整版

您看哪个方案更合适？
```

**选项B：部分交付**
- 优先完成核心功能/页面
- 交付时说明："这是核心部分，[其他部分]将在[X天内]补充"

**选项C：寻求帮助**
- 找同行协助（支付部分费用）
- 向客户说明情况，共同想办法

**不要做的**：
- ❌ 硬撑通宵，质量差还伤身体
- ❌ 不沟通直接延期
- ❌ 交付半成品不说明

**长期预防**：
- [项目时间规划指南](../01-生存期/基础保护/项目时间规划指南.md)
- [缓冲时间设置](../01-生存期/基础保护/合同审查清单.md)

### 情况2：客户突然要求大改

**紧急程度**：🟡 重要

**立即行动**：

**第一步：判断是否在范围内**
- 查看合同中的"修改范围"条款
- 如果是合同范围内：协商加急费用
- 如果是合同范围外：明确告知需要额外付费

**第二步：沟通话术**
```
[客户称呼]，收到修改需求。

我需要先确认：这些修改是在原合同范围内，还是新增需求？

如果是新增需求，根据工作量，需要：
- 额外[X天]时间
- 或额外[X元]费用

请先确认，我再安排工作。
```

**第三步：保护自己**
- 所有修改要求必须书面确认（邮件/微信文字）
- 不要口头承诺
- 修改前确认费用和时间

**长期预防**：
- [合同修改条款模板](../01-生存期/基础保护/合同审查清单.md)
- [需求变更管理](../02-成长期/项目管理/需求变更管理.md)

### 情况3：客户不满意，要求重做

**紧急程度**：🟡 重要

**立即行动**：

**第一步：了解具体原因**
- "具体哪些部分不满意？"
- "是不符合需求，还是审美不同？"
- "有参考案例吗？"

**第二步：判断责任**

**如果是你的责任**（没理解需求/质量不达标）：
- 承认问题
- 提出补救方案
- 适当补偿（如折扣/免费修改）

**如果是客户原因**（需求不清/审美变化）：
- 礼貌指出
- 提供两个方案：按原需求/按新需求（额外付费）
- 让客户选择

**第三步：沟通话术**
```
[客户称呼]，理解您的不满。

为了更好地解决，我想确认：
1. 主要不满意的点是？
2. 是不符合最初需求，还是需求有变化？

了解清楚后，我会提供解决方案。
```

**长期预防**：
- [需求确认清单](../01-生存期/基础保护/需求确认清单.md)
- [里程碑确认机制](../02-成长期/项目管理/里程碑确认.md)

### 情况4：客户拖欠款项

**紧急程度**：🟡 重要（超过30天）/ 🔴 紧急（超过60天）

**立即行动**：

**第一步：友好提醒（逾期7天内）**
```
[客户称呼]，您好。

[项目名称]的尾款[X元]，按合同应在[日期]支付。

可能您比较忙忘记了，烦请安排支付。

感谢！
```

**第二步：正式催收（逾期30天内）**
```
[客户称呼]，

关于[项目名称]的尾款[X元]，已逾期[X天]。

请本周内安排支付，否则我将：
1. 暂停当前进行中的工作
2. 按合同收取滞纳金
3. 必要时采取法律手段

希望友好解决。
```

**第三步：强硬措施（逾期60天+）**
- 发送律师函
- 公开曝光（谨慎使用）
- 法律诉讼

**长期预防**：
- [合同付款条款模板](../01-生存期/基础保护/合同审查清单.md)

### 情况5：客户要求降价/扣款

**紧急程度**：🟡 重要

**立即行动**：

**第一步：了解原因**
- "是因为预算问题，还是对成果不满意？"
- "是整体降价，还是部分扣款？"

**第二步：判断合理性**

**如果是你的责任**（质量不达标/延期）：
- 适当让步是合理的
- 但要有底线（最多10-20%）

**如果是客户原因**（预算不足/想占便宜）：
- 坚持合同价格
- 提供分期付款方案
- 必要时放弃这个客户

**第三步：谈判话术**
```
[客户称呼]，理解您的考虑。

按合同，价格是[X元]。如果您有困难，我可以提供：
1. 分期付款方案
2. 适当优惠[X%]（如果合理）

但大幅降价我无法接受，这会影响项目质量。
```

**长期预防**：
- [价值定价指南](../01-生存期/基础保护/报价定价指南.md)
- [谈判技巧](../02-成长期/客户开发/客户开发行动指南.md)

### 情况6：客户拒绝付款

**紧急程度**：🔴 紧急

**立即行动**：

**第一步：了解原因**
- "是对成果不满意，还是其他原因？"
- "具体哪些部分不满意？"

**第二步：尝试解决**
- 如果是合理不满：修改到满意为止
- 如果是无理拒付：保留证据，准备法律手段

**第三步：法律手段**
- 发送正式催款函
- 咨询律师
- 小额诉讼（如果金额不大）

**保留证据**：
- 合同
- 交付记录
- 沟通记录
- 工作过程文件

### 情况7：合同有陷阱，已经签了

**紧急程度**：🟡 重要

**立即行动**：

**第一步：评估风险**
- 陷阱条款具体是什么？
- 最坏情况是什么？
- 概率有多大？

**第二步：尝试协商**
- 以"补充协议"形式修改
- 强调"为了双方权益"
- 提供修改建议

**第三步：保护自己**
- 所有交付留证据
- 严格按合同执行
- 避免给对方把柄

**话术**：
```
[客户称呼]，为了更好地执行项目，建议我们签一个补充协议，明确：
1. [具体条款]
2. [具体条款]

这样双方都清楚，避免后续误会。
```

**长期预防**：
- [合同审查清单](../01-生存期/基础保护/合同审查清单.md)
- [律师审核](../01-生存期/基础保护/合同审查清单.md)

### 情况8：客户威胁要起诉

**紧急程度**：🔴 紧急

**立即行动**：

**第一步：不要慌**
- 大部分威胁只是威胁
- 真正起诉的很少
- 保持冷静，收集证据

**第二步：评估情况**
- 你有违约吗？
- 客户有证据吗？
- 最坏结果是什么？

**第三步：寻求专业帮助**
- 咨询律师
- 不要自己乱回复
- 让律师帮你回复

**第四步：准备应诉**
- 收集所有证据
- 准备答辩
- 必要时反诉

**保留证据**：
- 合同
- 所有沟通记录
- 交付物
- 付款记录

**长期预防**：
- [法律保护指南](../01-生存期/基础保护/合同审查清单.md)

### 情况9：知识产权纠纷

**紧急程度**：🟡 重要

**立即行动**：

**情况A：被指控侵权**
- 立即停止相关行为
- 评估是否确实侵权
- 寻求法律建议
- 准备和解或应诉

**情况B：发现被侵权**
- 收集证据
- 发律师函
- 要求停止侵权+赔偿
- 必要时起诉

**情况C：客户要求转让版权**
- 查看合同约定
- 额外收费（版权转让费）
- 明确授权范围

### 情况10：客户失联了

**紧急程度**：🟡 重要（项目进行时）/ 🟢 一般（项目结束后）

**立即行动**：

**第一步：多渠道联系**
- 微信
- 邮件
- 电话
- 共同认识的人

**第二步：设定等待期**
- 项目进行中：等待3-5天
- 项目结束后：等待7-14天

**第三步：最后通牒**
```
[客户称呼]，

多次联系未获回复。如果[X天内]仍未回复，我将：
1. 暂停项目
2. 按已完成部分结算
3. 保留采取进一步措施的权利

请尽快联系。
```

**第四步：止损**
- 暂停工作（避免继续投入）
- 按已完成部分结算
- 保留证据

**长期预防**：
- [定期沟通机制](../02-成长期/客户开发/客户开发行动指南.md)

### 情况11：客户要求无限修改

**紧急程度**：🟡 重要

**立即行动**：

**第一步：查看合同**
- 修改次数限制？
- 修改范围定义？

**第二步：明确告知限制**
```
[客户称呼]，

按合同，修改次数为[X次]，目前已使用[X次]。

如需继续修改，将按[X元/小时]或[X元/次]收费。

或者我们可以重新明确需求，避免反复修改。
```

**第三步：引导明确需求**
- "请一次性列出所有修改点"
- "请确认这是最终版本吗？"
- 修改后让客户书面确认

### 情况13：burnout，完全不想工作

**紧急程度**：🔴 紧急（身心健康优先）

**立即行动**：

**第一步：停止工作**
- 立即暂停所有项目
- 通知客户需要延期（不必说原因）
- 优先恢复身心健康

**第二步：紧急处理**
- 评估哪些项目必须完成
- 寻求帮助（朋友/同行协助）
- 与客户协商延期

**第三步：恢复计划**
- 休息1-2周
- 每天只做最基本的事
- 寻求专业帮助（心理咨询）

**沟通话术**：
```
[客户称呼]，

因个人健康原因，我需要暂停工作[X周]。

您的项目我会：
1. [安排：延期/转交他人/其他方案]
2. [补偿措施]

感谢您的理解。
```

**长期恢复**：
- [ burnout 恢复指南](../02-成长期/精力管理/burnout恢复指南.md)
- [工作边界设定](../02-成长期/精力管理/工作边界设定.md)

### 情况14：焦虑到无法工作

**紧急程度**：🔴 紧急

**立即行动**：

**第一步：缓解焦虑**
- 深呼吸（4-7-8呼吸法）
- 离开电脑，走动一下
- 做5分钟冥想

**第二步：分解任务**
- 把大任务分解成小任务
- 只做下一件最小的事
- 完成一个，再下一个

**第三步：寻求帮助**
- 和朋友/家人聊聊
- 寻求心理咨询
- 必要时就医

**第四步：调整期望**
- 今天只做60分
- 允许自己不完美
- 先完成，再完美

**紧急联系**：
- 心理援助热线：12355（青少年）/ 12320（卫生热线）
- 紧急情况下请拨打120或前往医院

## ❓ 你的情况不在这15种里？

**使用 AI 应急助手**：

复制下方的提示词，描述你的具体情况，让 AI 帮你分析：

```
你是我的 CCOS 应急顾问。我遇到了一个紧急情况，但不在应急手册的15种情况里。

**我的情况**：
[详细描述你的情况]

**请帮我**：
1. 评估紧急程度（🔴紧急/🟡重要/🟢一般）
2. 分析根本原因
3. 给出3个解决方案
4. 提供具体的沟通话术
5. 指出我需要避免的错误

**要求**：
- 先安抚情绪，再解决问题
- 方案要具体可操作
```

**或者寻求帮助**：
- 在 CCOS 社区提问
- 找有经验的同行咨询
- 必要时寻求专业帮助（律师/心理咨询师等）

**立即行动**：

**第一步：评估病情**
- 能工作吗？
- 需要休息多久？
- 需要就医吗？

**第二步：优先就医**
- 健康第一
- 工作可以延期
- 客户可以理解

**第三步：通知客户**

## 🤖 AI应急助手

**让 AI 帮你处理紧急情况**

### 使用方法

1. **打开 AI 工具**
2. **复制下方提示词**
3. **描述你的情况**
4. **获得即时建议**

### 通用应急提示词

```
你是我的 CCOS 应急顾问。我遇到了一个紧急情况，请帮我：

**我的情况**：
[描述具体情况]

**请帮我**：
1. 评估紧急程度（🔴紧急/🟡重要/🟢一般）
2. 给出3个解决方案（立即行动/短期应对/长期预防）
3. 提供具体的沟通话术
4. 指出我需要避免的错误

**要求**：
- 先安抚情绪，再解决问题
- 方案要具体可操作
- 考虑最坏情况和最好情况
```

**项目延期**：
```
你是我的项目顾问。 deadline 临近，我做不完了。

项目进度：[X]%

 deadline ：[日期]
客户性格：[好说话/严格/不确定]

请帮我：

2. 写延期申请话术
3. 提供补救方案
4. 给出长期预防建议
```

**款项催收**：
```
你是我的财务顾问。客户拖欠款项[X元]，已逾期[X天]。

已尝试：[你做过什么]
客户反应：[客户怎么说]
关系重要性：[重要/一般/不重要]

请帮我：
1. 制定催收策略（温和/正式/强硬）
2. 写催收话术
3. 评估法律手段可行性
4. 给出预防措施
```

**客户冲突**：
```
你是我的沟通顾问。我和客户发生了冲突。

冲突原因：[描述]
客户态度：[生气/冷漠/威胁]
我的诉求：[想要什么结果]

请帮我：
```
