# 状态检查使用指南

**每月10分钟，掌握你的职业健康状况**

> 就像体检一样，定期检查才能早发现问题。

---

## 🚀 推荐方式：AI对话式检查（最简单）

**不用填表，不用计算，像聊天一样完成状态检查。**

### 使用方法（30秒）

1. **打开AI工具**（ChatGPT/Claude/文心一言）
2. **复制[AI状态检查提示词](AI状态检查提示词.md)**
3. **粘贴给AI**
4. **回答AI的问题**
5. **获得完整报告**

**优势**：
- ✅ 自然对话，不用填表
- ✅ AI自动计算和评估
- ✅ 获得个性化报告和建议
- ✅ 可以随时追问和深入

## 📝 传统方式：手动填写清单

如果你更喜欢手动填写，也可以使用传统方式：

### 第一步：打开状态检查清单（1分钟）

打开 [状态检查清单](状态检查清单.md)

### 第二步：逐项检查并打分（3分钟）

**评分标准**：
- **5分**：完全达标，状态优秀
- **4分**：基本达标，状态良好
- **3分**：勉强达标，需要关注
- **2分**：不达标，需要改进
- **1分**：严重不达标，立即处理

**快速打分技巧**：
- 不要过度思考，凭直觉打分
- 如果不确定，打3分
- 诚实面对，不要美化

### 第三步：计算总分并判定阶段（1分钟）

**计算公式**：
```
总分 = 生存指标×0.4 + 成长指标×0.3 + 健康指标×0.2 + 风险指标×0.1
```

**权重说明**：
- **生存指标 40%**：最基础，生存问题是首要问题
- **成长指标 30%**：长期发展，决定职业上限
- **健康指标 20%**：可持续发展，避免 burnout
- **风险指标 10%**：预防性，防患于未然

**可以调整吗？**
- 可以！根据你的阶段调整权重
- 生存期：生存60%，成长20%，健康10%，风险10%
- 成长期：生存30%，成长50%，健康15%，风险5%
- 影响力期：生存20%，成长30%，健康30%，风险20%

**阶段判定**：
- **生存期**：总分 < 2.5 或 生存指标 < 3
- **成长期**：2.5 ≤ 总分 < 3.5 且 生存指标 ≥ 3
- **影响力期**：总分 ≥ 3.5
- **风险期**：健康指标 < 2 或 风险指标 < 2

**完成！** 你已经完成了本月的状态检查。🎉

## 📊 状态解读指南

### 生存指标解读

| 得分 | 状态 | 含义 | 行动建议 |
|:---:|:---:|:---|:---|
| 4分 | 🟢 稳固 | 生存无忧，基础扎实 | 保持现状，关注成长 |
| 3分 | 🟡 基本 | 基本安全，但有隐患 | 加强客户开发，增加储备 |
| 2分 | 🔴 风险 | 生存有风险 | 优先解决收入问题，减少支出 |
| 1分 | 🚨 危险 | 生存危机 | 立即行动，寻求紧急收入来源 |

**关键问题**：
- 如果月收入不稳定 → 立即开始客户开发
- 如果应急储备不足 → 每月强制储蓄
- 如果客户来源单一 → 拓展获客渠道

---

### 成长指标解读

| 得分 | 状态 | 含义 | 行动建议 |
|:---:|:---:|:---|:---|
| 4分 | 🟢 良好 | 成长势头好 | 加速成长，突破瓶颈 |
| 3分 | 🟡 稳步 | 稳步成长 | 保持节奏，寻找突破点 |
| 2分 | 🔴 停滞 | 成长停滞 | 学习新技能，尝试新方向 |
| 1分 | 🚨 倒退 | 能力倒退 | 重新规划，寻找导师 |

**关键问题**：
- 如果单价长期不提升 → 学习价值定价
- 如果没有个人品牌 → 开始内容创作
- 如果没有学习新技能 → 制定学习计划

---

### 健康指标解读

| 得分 | 状态 | 含义 | 行动建议 |
|:---:|:---:|:---|:---|
| 4分 | 🟢 健康 | 身心健康 | 保持，预防 burnout |
| 3分 | 🟡 基本 | 基本健康 | 增加休息，调整节奏 |
| 2分 | 🔴 预警 | 健康预警 | 立即调整，减少工作 |
| 1分 | 🚨 危险 | 健康危险 | 停止工作，优先恢复 |

**关键问题**：
- 如果工作时长过长 → 设定边界，学会拒绝
- 如果没有运动 → 每周至少3次，每次30分钟
- 如果睡眠质量差 → 改善睡眠环境，建立睡前习惯

---

### 风险指标解读

| 得分 | 状态 | 含义 | 行动建议 |
|:---:|:---:|:---|:---|
| 4分 | 🟢 安全 | 无风险 | 保持警惕，定期监控 |
| 3分 | 🟡 轻微 | 轻微风险 | 及时处理，防止恶化 |
| 2分 | 🔴 较高 | 风险较高 | 立即处理，制定预案 |
| 1分 | 🚨 严重 | 严重风险 | 紧急处理，寻求帮助 |

**关键问题**：
- 如果有客户拖欠 → 立即催收，必要时法律手段
- 如果有项目超期 → 与客户沟通，调整预期
- 如果有身体预警 → 立即休息，就医检查

## 🎯 基于结果的行动建议

### 场景1：生存期（总分<2.5）

**特征**：
- 收入不稳定或不足
- 客户来源单一
- 应急储备不足

**优先级行动**：

**第1周**：
- [ ] 列出所有潜在客户，开始客户开发
- [ ] 审查现有合同，确保收款
- [ ] 计算最低生活成本，制定预算

**第2-4周**：
- [ ] 每天投入2小时客户开发
- [ ] 接受合理价格的紧急项目
- [ ] 建立应急储备计划（每月存20%收入）

**本月目标**：
- 新增1-2个客户
- 收入覆盖基本支出
- 建立3个月应急储备计划

---

### 场景2：成长期（2.5≤总分<3.5）

**特征**：
- 生存基本稳定
- 想提升单价和影响力
- 有学习意愿

**优先级行动**：

**第1周**：
- [ ] 分析现有客户，找出高价值客户特征
- [ ] 研究市场价，确定目标单价
- [ ] 更新作品集和个人品牌

**第2-4周**：
- [ ] 开始内容创作（每周2-3篇）
- [ ] 学习一项新技能
- [ ] 尝试提升10-20%报价

**本月目标**：
- 发布8-12篇内容
- 完成1个技能学习
- 新客户报价提升10%

---

### 场景3：影响力期（总分≥3.5）

**特征**：
- 收入稳定且较高
- 有个人品牌
- 想实现自由和规模化

**优先级行动**：

**第1周**：
- [ ] 评估被动收入可能性
- [ ] 整理可产品化的知识/作品
- [ ] 设计理想生活图景

**第2-4周**：
- [ ] 开始第一个被动收入项目
- [ ] 建立团队协作流程
- [ ] 规划长期财务目标

**本月目标**：
- 完成被动收入MVP
- 建立1个自动化流程
- 制定3年财务自由计划

---

### 场景4：风险期（健康<2 或 风险<2）

**特征**：
- 身体或心理出现预警
- 有紧急风险需要处理
- 可能面临 burnout

**紧急行动**：

**立即（今天）**：
- [ ] 停止所有非紧急工作
- [ ] 评估风险严重程度
- [ ] 寻求帮助（朋友/专业人士）

**本周**：
- [ ] 处理最紧急的风险（拖欠款/超期项目）
- [ ] 每天保证8小时睡眠
- [ ] 每天30分钟运动

**本月**：
- [ ] 减少工作时长至40小时/周
- [ ] 建立健康习惯
- [ ] 重新评估职业方向

## 💡 高效使用技巧

### 技巧1：诚实面对

**不要**：
- 美化分数
- 回避问题
- 自我欺骗

**要**：
- 真实打分
- 直面问题
- 寻求帮助

### 技巧2：关注趋势

**不要**：
- 只看单月分数
- 过度反应单次波动
- 忽视长期趋势

**要**：
- 对比3个月趋势
- 关注改善方向
- 庆祝进步

### 技巧3：小步改进

**不要**：
- 制定宏大计划
- 一次性改变太多
- 追求完美

**要**：
- 每月改进1-2点
- 循序渐进
- 先完成再完美

### 技巧4：寻求支持

**方式**：
- 找 accountability partner（问责伙伴）
- 加入CCOS社区分享
- 定期与导师/朋友讨论

### 技巧5：灵活调整

**原则**：
- 清单是工具，不是枷锁
- 根据情况调整权重
- 保持核心指标不变

## 🔗 与其他复盘系统的关系

### 状态检查 vs 每日循环

| | 状态检查 | 每日循环 |
|:---|:---|:---|
| **频率** | 每月1次 | 每天1-2次 |
| **时长** | 10分钟 | 15分钟 |
| **重点** | 战略层面（4大指标） | 战术层面（今日任务） |
| **关系** | 月度体检 | 日常保养 |

**如何结合**：
- 每日循环发现问题 → 状态检查确认趋势
- 状态检查制定方向 → 每日循环执行落地

### 状态检查 vs 周复盘

| | 状态检查 | 周复盘 |
|:---|:---|:---|
| **频率** | 每月1次 | 每周1次 |
| **重点** | 职业健康（4大指标） | 项目进展（客户/收入） |
| **关系** | 健康诊断 | 业务盘点 |

**如何结合**：
- 周复盘发现业务问题 → 状态检查评估影响
- 状态检查健康预警 → 周复盘调整工作负荷

### 状态检查 vs 月复盘

| | 状态检查 | 月复盘 |
|:---|:---|:---|
| **频率** | 每月1次 | 每月1次 |
| **重点** | 职业状态（健康度） | 业务成果（财务/项目） |
| **关系** | 过程指标 | 结果指标 |

**如何结合**：
- 一起做（每月最后一天）
- 状态检查（10分钟）→ 月复盘（30分钟）
- 健康度影响业务成果，业务成果反映健康度

## ⚠️ 常见错误

### 错误1：过度关注分数

**表现**：
- 为了高分而美化
- 分数低就沮丧
- 忽视实际改善

**解决**：
- 分数只是参考
- 关注趋势而非绝对值
- 重视行动而非分数

### 错误2：检查后无行动

**表现**：
- 检查完就放下
- 不制定行动计划
- 下个月还是一样

**解决**：
- 检查后必须制定行动计划
- 把行动加入每日/每周任务
- 下次检查回顾行动结果

### 错误3：检查太频繁或太少

**表现**：
- 每周都检查（太频繁）
- 半年才检查一次（太少）

**解决**：
- 每月1次是最佳频率
- 紧急情况可增加1次
- 设置提醒确保不忘记

### 错误4：独自检查，不分享

**表现**：
- 不告诉任何人
- 不寻求建议
- 陷入自我封闭

**解决**：
- 找 accountability partner
- 在CCOS社区分享
- 定期与导师讨论
