## 🖨️ 打印方法

### 方法1：浏览器打印（推荐）

**步骤**：
1. 打开你想打印的文档
2. 按 `Ctrl+P`（Windows）或 `Cmd+P`（Mac）
3. 在打印设置中调整选项（见下方"打印设置建议"）
4. 点击打印

**优点**：
- 最简单直接
- 保留原文档结构

**缺点**：
- 可能包含不必要的元素（导航、链接等）
- 代码块和表格可能显示不全

### 方法2：生成PDF（推荐用于iPad/离线阅读）

**步骤**：
1. 打开你想打印的文档
2. 按 `Ctrl+P`（Windows）或 `Cmd+P`（Mac）
3. 选择"目标打印机"为"另存为PDF"
4. 保存到电脑
5. 传输到iPad或手机

**优点**：
- 可以离线阅读
- 保留格式
- 便于携带

**工具推荐**：
- **Windows**：Microsoft Print to PDF（系统自带）
- **Mac**：系统自带的"另存为PDF"
- **Chrome浏览器**：打印时选择"另存为PDF"

### 方法3：使用Markdown转PDF工具

**推荐工具**：

| 工具 | 平台 | 特点 |
|:---|:---|:---|
| **Typora** | Win/Mac | 导出PDF格式最好 |
| **VS Code + Markdown PDF插件** | 全平台 | 免费，可批量转换 |
| **Pandoc** | 全平台 | 命令行工具，功能强大 |

**使用步骤（Typora为例）**：
1. 下载并安装 [Typora](https://typora.io/)
2. 打开Markdown文件
3. 文件 → 导出 → PDF
4. 调整导出设置
5. 保存PDF

## ⚙️ 打印设置建议

### 页面设置

**纸张大小**：
- A4（推荐，最常用）
- Letter（美国标准）

**方向**：
- 纵向（推荐，适合阅读）
- 横向（仅用于宽表格）

**边距**：
- 默认边距即可
- 如果想省纸，可以选择"窄边距"

### 内容设置

**页眉/页脚**：
- ✅ 显示页码（方便查阅）
- ❌ 隐藏URL（链接显示为完整URL会占用大量空间）
- ❌ 隐藏标题（节省空间）

**背景图形**：
- ❌ 不打印背景（省墨）

**缩放**：
- 选择"适合页面宽度"或100%

### 浏览器特定设置

**Chrome浏览器**：
1. 更多设置 →
2. ✅ 页眉和页脚（可选）
3. ❌ 背景图形
4. 缩放：100%

**Firefox浏览器**：
1. 页面设置 →
2. 页眉和页脚：自定义，删除URL
3. 缩放：100%

**Safari浏览器**：
1. 文件 → 打印 →
2. 显示详细信息
3. 页眉/页脚：自定义

## 📋 打印优化技巧

### 技巧1：选择性打印

**不要打印整个仓库！**

只打印你需要的部分：
- 应急手册（完整打印）
- 常用模板（选择性打印）
- 检查清单（打印后勾选使用）

### 技巧2：双面打印

**省纸技巧**：
- 使用双面打印（如果打印机支持）
- 或手动双面：先打印奇数页，翻转后再打印偶数页

### 技巧3：多页合一

**省纸技巧**：
- 在打印设置中选择"多页合一"
- 2页或4页合一（适合检查清单）

### 技巧4：黑白打印

**省墨技巧**：
- 选择黑白/灰度打印
- 除非有彩色图表，否则不需要彩色

### 技巧5：环保打印

**环保建议**：
- ♻️ 使用再生纸
- ♻️ 双面打印（省纸50%）
- ♻️ 多页合一（省纸75%）
- ♻️ 废纸回收（打印错误的纸张可以回收）
- ♻️ 定期整理，只保留最常用的（减少重复打印）

### 技巧6：预览检查

**打印前必做**：
1. 先打印预览，检查格式
2. 确保表格和代码块显示完整
3. 检查页码是否连续

## 📱 替代方案：离线阅读

**如果你不想打印，但需要离线访问**：

### 方案1：保存网页

**步骤**：
1. 在浏览器中打开文档
2. 按 `Ctrl+S`（Windows）或 `Cmd+S`（Mac）
3. 选择保存类型："网页，完整"
4. 保存到电脑

**优点**：保留格式和图片

### 方案2：使用阅读列表

**Safari浏览器**：
1. 打开文档
2. 点击分享按钮 → "添加到阅读列表"
3. 离线时可以阅读

**Chrome浏览器**：
1. 安装 "Read Later" 扩展
2. 保存页面到阅读列表

### 方案3：GitHub App离线模式

**GitHub App** 支持离线阅读：
1. 在有网络时打开文档
2. 文档会自动缓存
3. 离线时可以继续阅读

## 🆘 打印问题排查

### 问题1：表格显示不全

**原因**：表格太宽，超出页面

**解决**：
- 尝试横向打印
- 或使用Markdown转PDF工具

### 问题2：代码块被截断

**原因**：代码行太长

**解决**：
- 使用Markdown转PDF工具
- 或手动复制代码块到新文档

### 问题3：链接显示为长URL

**原因**：浏览器默认设置

**解决**：
- 在打印设置中隐藏页眉/页脚
- 或使用Markdown转PDF工具

### 问题4：图片不显示

**原因**：网络问题或设置问题

**解决**：
- 确保网络连接正常
- 在打印设置中勾选"打印背景图形"
