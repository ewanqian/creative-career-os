## ❓ 常见问题（FAQ）

### 使用问题

**Q1：我是新手，应该从哪里开始？**

**A**：
1. 先查看 [新用户引导](00-新用户引导.md)
2. 使用 [对话式职业评估](对话式职业评估.md) 判定你的阶段
3. 根据你的阶段，查看对应指南（生存期/成长期/影响力期）
4. 收藏 [工具库索引](工具库索引.md)，需要时快速查找

---

**Q2：CCOS适合我吗？**

**A**：CCOS 适合：
- ✅ 自由职业者（设计师、插画师、摄影师、写作者等）
- ✅ 创意工作者（独立开发者、手工艺人、音乐制作人等）
- ✅ 独立从业者（咨询师、培训师、教练等）

如果你：
- 想系统化职业发展
- 希望从生存到影响力
- 愿意持续学习和迭代

那么 CCOS 适合你。

---

**Q3：我需要付费吗？**

**A**：CCOS 完全免费：
- ✅ 所有内容免费使用
- ✅ 可以自由修改和分享
- ✅ 采用 CC BY-NC-SA 4.0 开源协议

**唯一限制**：不得用于商业售卖。

---

**Q4：如何找到适合我行业的指导？**

**A**：
1. 查看 [行业适配指南](行业适配指南.md)
2. 找到你的行业
3. 阅读行业特定的适配方法
4. 参考行业案例

如果你的行业还没有覆盖，可以：
- 参考相似行业的方法
- 在Issues中提出需求
- 贡献你的经验

### 技术问题

**Q11：使用CCOS需要什么系统要求？**

**A**：
- **浏览器**：Chrome、Firefox、Safari、Edge 最新版本
- **网络**：需要网络连接（离线使用需先下载）
- **设备**：电脑、平板、手机均可
- **GitHub账号**：如需贡献内容，需要GitHub账号

**推荐配置**：
- 使用Chrome或Safari浏览器，兼容性最好
- 在GitHub上浏览，体验最佳

---

**Q12：如何下载整个CCOS？**

**A**：
```bash
# 使用git克隆
git clone https://github.com/ewanqian/creative-career-os.git

# 或者下载ZIP
# 点击GitHub页面的 "Code" → "Download ZIP"
```

---

**Q13：如何离线使用？**

**A**：
1. 下载整个仓库到本地
2. 使用Markdown阅读器打开
3. 或生成PDF（查看 [打印指南](打印指南.md)）

---

**Q14：文档中的链接打不开怎么办？**

**A**：
1. 确保网络连接正常
2. 如果是相对路径链接，在GitHub上打开
3. 如果链接失效，请在Issues中报告

## 🤝 社区准则

### 我们的承诺

- 🌱 **友善互助**：友好对待每一位成员
- 🔍 **求真务实**：基于事实和经验分享
- ♻️ **持续学习**：保持开放心态，持续成长
- 🤝 **互助共赢**：帮助他人就是帮助自己

### 不可接受的行为

- ❌ 人身攻击或骚扰
- ❌ 歧视性言论
- ❌ 垃圾信息或广告

- ❌ 其他不专业行为

### 执行

违反社区准则可能导致：
- 警告
