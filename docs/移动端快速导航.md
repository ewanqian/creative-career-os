## 💡 移动端使用技巧

### 技巧1：收藏常用页面

把最常用的3-5个页面添加到手机浏览器书签：
1. 本页面（移动端快速导航）
2. 应急手册
3. 工具库索引

### 技巧2：使用GitHub App

下载 GitHub 官方 App，体验更好：
- [iOS 下载](https://apps.apple.com/app/github/id1477376905)
- [Android 下载](https://play.google.com/store/apps/details?id=com.github.android)

**💡 提示**：GitHub App 支持夜间模式，在手机上阅读更舒适

### 技巧3：使用AI语音输入

在手机上使用AI时：
- 使用语音输入代替打字
- 长按输入框选择"语音输入"
- 说普通话，AI识别率高

### 技巧4：离线访问

**下载PDF版本**（即将推出）：
- 完整版PDF（适合深度阅读）
- 精简版PDF（适合快速查阅）
- 应急手册PDF（紧急情况离线使用）

**临时方案**：
- 在电脑上打开需要的页面
- 使用浏览器"添加到主屏幕"功能
- 或使用"阅读列表"保存离线阅读

### 技巧5：快速搜索

在GitHub页面使用搜索：
1. 点击页面顶部的搜索框
2. 输入关键词（如"报价"、"合同"）
3. 选择"在当前仓库搜索"

## 📱 移动端优化提示

### 表格显示优化

原表格在手机上可能显示不全，本页面已优化为：
- 使用列表代替大表格
- 每个工具单独一行
- 点击区域足够大

### 长文档处理

长文档在手机上阅读困难：
- 使用"目录"快速跳转
- 分段阅读，不要一次看完
- 重要的内容先收藏

### 网络不稳定时

地铁/电梯等网络不稳定场景：
- 提前加载页面
- 使用浏览器"阅读模式"
- 截图保存重要内容

## 🤖 AI辅助（移动端推荐）

### 推荐AI工具（手机端）

| 工具 | iOS | Android | 特点 |
|:---|:---:|:---:|:---|
| **ChatGPT** | ✅ | ✅ | 支持语音输入，体验最好 |
| **Claude** | 网页 | 网页 | 逻辑清晰，适合深度分析 |
| **文心一言** | ✅ | ✅ | 国内可用，中文好 |
| **Kimi** | ✅ | ✅ | 长文本处理，适合快速检查 |

### 移动端AI使用技巧

1. **语音输入**：长按输入框，选择语音输入
2. **复制提示词**：在本页面长按提示词，复制后粘贴到AI工具
3. **分次对话**：长对话可以分多次进行，AI会记住上下文
4. **保存对话**：重要的对话结果截图或复制保存
