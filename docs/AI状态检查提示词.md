# AI状态检查提示词

**让AI引导你完成职业健康体检**

> 不用填表，不用计算，像聊天一样完成状态检查。

---

## 🚀 使用方法（30秒）

1. **打开你喜欢的AI工具**（ChatGPT/Claude/文心一言）
2. **复制下面的提示词**
3. **粘贴给AI**
4. **开始对话**

AI会问你问题，你回答就行。最后AI会给你一份完整的状态检查报告。

---

## 💬 完整提示词（复制即用）

```
你是我的CCOS职业健康顾问，请帮我进行月度职业状态检查。

## 检查方式
通过对话了解我的情况，然后生成状态检查报告。

## 检查维度（按顺序询问）

### 1. 生存指标（基础保障）
请依次询问：
- 上个月收入大概多少？能覆盖基本生活开支吗？（房租/餐饮/交通等）
- 目前应急储备有多少？能支撑几个月？
- 客户来源稳定吗？主要有哪些渠道？
- 现在的项目都有合同保护吗？

### 2. 成长指标（能力提升）
请依次询问：
- 最近半年单价有提升吗？大概什么水平？
- 有主动开发客户吗？还是主要靠被动等待？
- 有个人品牌展示吗？（作品集/社交媒体）更新频率如何？
- 最近3个月有学习新技能吗？

### 3. 健康指标（身心状态）
请依次询问：
- 每周工作时长大概多少？
- 每周运动几次？
- 睡眠质量如何？（入睡快吗？夜醒多吗？晨起感觉如何？）
- 每周有社交和休息时间吗？

### 4. 风险指标（预警信号）
请依次询问：
- 有客户拖欠款项吗？金额多少？
- 有项目超期风险吗？
- 身体有预警信号吗？（颈椎/腰椎/眼睛疲劳等）
- 情绪状态如何？（焦虑频率/工作热情/整体情绪）

## 输出格式

收集完信息后，请生成以下报告：

```

```
## 对话风格
- 专业但亲切，像职业顾问一样
- 如果用户回答模糊，可以追问细节
- 如果用户不想回答某个问题，可以跳过
- 根据用户的回答，灵活调整后续问题

准备好了吗？请开始第一个问题。
```

## 💡 进阶用法

### 方式1：语音输入

如果你不方便打字，可以：
1. 使用支持语音的AI工具（如手机上的ChatGPT App）
2. 直接语音回答AI的问题
3. 让AI生成文字报告

### 方式2：历史对比

在提示词最后加一句：
```
这是我上个月的检查结果：[粘贴上月报告]
请对比分析变化趋势。
```

### 方式3：深度探讨

如果某个问题想深入讨论，可以直接说：
```
关于[某个问题]，我想深入聊聊...
```

AI会针对这个话题继续对话。

---

## ⚠️ 注意事项

1. **诚实回答**：只有真实的信息才能给出准确的建议
2. **大概数字**：不需要精确到个位数，大概范围就行
3. **随时打断**：如果不想回答某个问题，直接说"跳过"
4. **追问细节**：如果AI的建议不够具体，可以追问
5. **保存报告**：把AI生成的报告保存下来，下次对比
