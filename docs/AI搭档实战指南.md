### 第二步：设置AI角色（2分钟）

复制以下提示词，作为对话开头：

```
你是我的CCOS AI搭档，扮演三个角色：

1. **导师**：帮助我成长，提供职业建议
2. **顾问**：分析具体问题，提供专业意见

- 职业：[设计师/插画师/文案等]
- 经验：[0-2年/2-5年/5年+]

- 当前挑战：[具体描述]
```

**示例开场白**：

```

根据我的评估，这个项目至少需要15000元。

请帮我：
1. 分析这个报价是否合理
2. 提供3个应对策略
3. 给出一个具体的回复话术
```

## 💬 提示词模板库

### 场景1：报价分析

**提示词模板**：

```
请帮我分析这个报价是否合理。

项目信息：
- 项目类型：[Logo设计/网站设计/文案等]
- 项目范围：[具体描述]
- 客户预算：[金额]
- 我的报价：[金额]
- 预计工时：[小时数]
- 我的时薪：[金额]

请分析：
1. 这个报价是否合理？（偏高/偏低/合理）
2. 客户预算低的可能原因
3. 3个应对策略
4. 具体的回复话术

参考市场价：[可选项，提供市场价格参考]
```

**示例对话**：

**用户**：
```
请帮我分析这个报价。

项目：一套品牌VI（Logo+名片+信纸+基础规范）
客户预算：8000元
我的报价：15000元
预计工时：80小时
我的时薪：200元

客户说预算有限，但项目很急。
```

**AI回答**（示例）：
```
## 报价分析

**合理性评估**：你的报价合理，客户预算偏低约47%。

**分析**：
- 按你的时薪200元，80小时 = 16000元
- 你的报价15000元已经是优惠价格
- 客户预算8000元仅覆盖50%成本

**客户预算低的可能原因**：
1. 对VI设计价值认知不足
2. 确实有预算限制（初创公司）
3. 试探你的底线

**3个应对策略**：

策略1：价值教育
- 向客户解释VI的长期价值
- 展示案例：好的VI如何帮助品牌增长

策略2：分期付款
- 接受8000元预算，但分阶段交付
- 先做Logo（5000元），后续再补充

策略3：调整范围
- 减少交付物，匹配预算
- 只做Logo+名片（8000元）

**回复话术**：
"理解您的预算考虑。基于项目范围，我的报价是15000元。

如果您预算有限，我建议两个方案：
1. 先做核心部分（Logo+名片），预算8000元
2. 分阶段进行，本月先做Logo

您看哪种方式更合适？"
```

### AI能做的（放心使用）

| 能力 | 示例 | 可靠度 |
|:---|:---|:---:|
| **信息整理** | 整理合同条款、项目需求 | ⭐⭐⭐⭐⭐ |
| **模板生成** | 生成邮件模板、报价单 | ⭐⭐⭐⭐⭐ |

| **学习辅导** | 解释概念、推荐学习资源 | ⭐⭐⭐⭐⭐ |
| **情绪支持** | 鼓励、心态调整建议 | ⭐⭐⭐⭐ |

### AI不能做的（不要依赖）

| 能力 | 原因 | 替代方案 |
|:---|:---|:---|
| **法律建议** | AI不是律师，可能出错 | 咨询专业律师 |
| **财务建议** | AI不是财务顾问 | 咨询专业财务顾问 |
| **最终决策** | AI不了解你的全部情况 | 参考AI建议，自己做决定 |
| **创意执行** | AI不能代替你创作 | AI提供灵感，你执行 |
| **客户关系** | AI不能代替你建立关系 | AI辅助沟通，你维护关系 |

### AI可能出错的（需要验证）

| 场景 | 风险 | 验证方法 |
|:---|:---|:---|
| **市场数据** | 数据可能过时 | 查证最新数据 |
| **法律法规** | 可能引用错误 | 查证官方文件 |
| **技术细节** | 可能有误 | 查证官方文档 |
| **价格建议** | 可能不符合实际 | 参考多个来源 |

## 🔍 AI回答质量评估

### 评估维度

| 维度 | 评估问题 | 评分 |
|:---|:---|:---:|
| **准确性** | 信息是否正确？ | 1-5 |
| **实用性** | 建议是否可操作？ | 1-5 |
| **完整性** | 是否覆盖了所有方面？ | 1-5 |
| **个性化** | 是否符合我的情况？ | 1-5 |
| **逻辑性** | 推理是否合理？ | 1-5 |

**总分**：25分
- 20-25分：优秀，可以直接使用
- 15-19分：良好，需要微调
- 10-14分：一般，需要大幅修改
- <10分：较差，重新提问

### 改进方法

如果AI回答质量不高：

1. **补充信息**：提供更多背景
2. **明确要求**：明确告诉AI你需要什么
3. **分步提问**：把复杂问题拆分成小问题
4. **换种问法**：用不同的方式提问
5. **换AI工具**：不同AI有不同优势

## 💡 高效使用技巧

### 技巧1：提供足够背景

**错误示例**：
```
帮我写一封开发信。
```

**正确示例**：
```
帮我写一封客户开发信。

我的背景：
- 职业：UI设计师
- 经验：3年
- 专长：APP界面设计
- 目标客户：互联网初创公司

客户背景：
- 公司：某电商APP
- 阶段：刚完成融资
- 需求：可能需要进行UI改版

请写一封专业但有亲和力的开发信。
```

### 技巧2：明确要求格式

**示例**：
```
请帮我分析这个项目。

要求输出格式：
1. 风险评估（高/中/低）
2. 3个应对策略
3. 推荐策略及理由
4. 具体行动步骤
```

### 技巧3：迭代优化

**第一轮**：获取初步回答
**第二轮**：要求深化某个点
**第三轮**：要求提供示例

**示例**：
```
第一轮：请帮我分析这个报价。
第二轮：请详细说明第一个策略。
第三轮：请给我一个具体的回复话术。
```

### 技巧4：保存优秀提示词

**方法**：
- 建立提示词库（Notion/飞书文档）
- 分类整理（报价/合同/沟通/复盘）
- 定期更新优化

### 技巧5：多AI对比

**方法**：
- 同一个问题问不同AI
- 对比回答质量
- 选择最好的或综合

## 🎯 AI使用频率建议

**强烈推荐使用**：
- ✅ 头脑风暴和创意发散

**每周**：
- 周复盘（15分钟）
- 项目分析（30分钟）

## 🧠 AI使用心理准备

### 常见担忧

**担忧1：AI会取代我的工作吗？**

**现实**：
- AI会取代部分重复性工作
- 但会创造新的机会（AI协作、AI管理）
- 创意核心（审美、情感、策略）难以被取代

**应对**：
- 把AI当作工具，不是威胁
- 专注于AI难以替代的能力
- 学习AI协作技能

**担忧2：用AI是不是"作弊"？**

**现实**：
- 用AI辅助和用计算器一样，是工具使用
- 最终作品质量取决于你的判断和修改
- 客户买的是你的专业能力和审美

**应对**：
- 明确AI是辅助工具
- 保持对最终作品的责任
- 不断提升核心能力

**担忧3：AI回答错了怎么办？**

**现实**：
- AI确实会出错（幻觉）
- 关键信息需要验证
- 最终决策要自己负责

**应对**：
- 对AI回答保持批判性思维
- 重要信息多方验证
- 建立自己的知识体系

### 健康心态

**正确心态**：
- AI是搭档，不是对手
- AI是杠杆，放大你的能力
- AI是教练，帮助你成长

**避免心态**：
- 完全依赖AI
- 害怕被AI取代
- 认为AI万能

## ⚠️ 常见错误

### 错误1：期望过高

**表现**：
- 认为AI可以解决所有问题
- 不验证AI的回答
- 完全依赖AI做决策

**解决**：
- AI是助手，不是替代
- 关键决策要自己判断
- 验证重要信息

### 错误2：提示词太模糊

**表现**：
- "帮我写个文案"
- "分析一下"

**解决**：
- 提供具体背景
- 明确输出要求
- 给出示例

### 错误3：不迭代

**表现**：
- 一次提问不满意就放弃
- 不追问细节

**解决**：
- 多轮对话
- 要求澄清
- 调整提示词

### 错误4：泄露敏感信息

**表现**：
- 把客户合同全文发给AI
- 泄露商业机密

**解决**：
- 脱敏处理
- 只提供必要信息
- 使用本地AI（如有敏感信息）
