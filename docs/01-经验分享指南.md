# 经验分享指南

## 什么是"经验分享"？

**经验分享**是 Creative-Career-OS 知识体系的源头。它指的是从业者遇到的真实、具体、已脱敏的职业困境描述。每一次分享都是一个知识矿点，经过社区讨论和提炼，最终转化为可复用的指南和工具。

## 为什么需要分享经验？

1. **帮助他人**：你的一次经历，可能成为他人避坑的明灯
2. **获得帮助**：通过社区讨论，获得多元视角的解决方案
3. **沉淀知识**：将个人经验转化为公共资产，实现价值最大化
4. **社区共建**：参与构建行业共用的职业支持系统

## 经验分享流程

### 第一步：准备你的故事

在分享之前，请先回忆并整理你的经历：
- 发生了什么？（事件背景）
- 问题出在哪里？（核心困境）
- 你当时如何应对？（你的行动）
- 结果如何？（最终结局）
- 如果重来，你会怎么做？（反思与建议）

### 第二步：彻底脱敏（最重要！）

**绝对红线**：所有可识别信息必须被替换！

#### 需要替换的信息类型：

| 信息类型 | 错误示例 | 正确示例 |
|:---|:---|:---|
| **公司名称** | 蓝图设计工作室、星辰广告公司 | [某设计工作室]、[某广告公司] |
| **人名** | 王经理、陈总 | [客户A]、[对接人B] |
| **项目名称** | 悦享咖啡品牌升级、春季促销海报 | [某咖啡品牌项目]、[某促销活动] |
| **具体金额** | 5万元、8000元 | [约五位数预算]、[约中四位数] |
| **具体时间** | 2024年3月15日 | [去年春季]、[约半年前] |
| **地点** | 杭州西湖区、深圳南山 | [某二线城市]、[某创意园区] |
| **作品名称** | 《晨光》系列、《都市印象》 | [某系列作品]、[某主题创作] |

#### 脱敏检查清单：

- [ ] 所有公司名称已替换为通用描述
- [ ] 所有人名已替换为代号（如[客户A]）
- [ ] 所有具体金额已模糊化（如[约五位数]）
- [ ] 所有具体时间已相对化（如[去年]）
- [ ] 所有具体地点已泛化（如[一线城市]）
- [ ] 所有作品名称已匿名化
- [ ] 所有联系方式、邮箱、电话已删除
- [ ] 检查是否有组合信息可推断出具体身份

### 第三步：使用标准模板

```markdown
## 经验分享标题：[一句话概括核心问题]

### 基本信息
- **分享编号**：[待分配]
- **分享日期**：[YYYY-MM-DD]
- **涉及领域**：[合同/报价/沟通/版权/其他]
- **职业身份**：[自由设计师/插画师/新媒体运营/其他]
- **项目类型**：[品牌设计/UI设计/插画/视频/其他]

### 事件背景
[描述项目的来龙去脉，包括：
- 如何接触到这个项目
- 客户的基本情况（已脱敏）
- 项目的初步范围和预期
- 当时的合作模式]

### 核心困境
[清晰描述遇到的具体问题：
- 问题是什么？
- 何时发现的？
- 问题的严重程度？
- 对你造成了什么影响？]

### 你的应对
[描述你当时采取的行动：
- 第一反应是什么？
- 尝试了哪些解决方案？
- 与客户的沟通过程？
- 最终结果如何？]

### 反思与教训
[总结经验教训：
- 如果重来，你会怎么做？
- 哪些信号应该提前警惕？
- 有什么预防建议？
- 希望获得社区什么帮助？]

### 讨论标签
#[合同问题] #[报价问题] #[版权纠纷] #[沟通困境] #[其他]
```

### 第四步：提交到 GitHub Issues

1. 访问 [Issues 页面](https://github.com/ewanqian/creative-career-os/issues)
2. 点击 "New issue"
3. 标题格式：`[经验分享] 一句话概括核心问题`
4. 粘贴你的分享内容
5. 添加合适的标签
6. 点击 "Submit new issue"

### 第五步：参与社区讨论

提交后，社区成员可能会：
- 询问细节（会提醒你注意脱敏）
- 分享类似经历
- 提供解决建议
- 讨论如何转化为指南

请积极参与讨论，这是价值最大的环节。

## 分享原则

### ✅ 鼓励分享的内容

- 真实的职业困境
- 具体的应对过程
- 明确的反思总结
- 可复用的经验教训

### ❌ 不建议分享的内容

- 纯情绪宣泄，没有建设性
- 过于笼统，缺乏具体细节
- 未脱敏，可能暴露隐私
- 法律纠纷正在进行中（避免法律风险）

## 隐私保护承诺

1. **绝对匿名**：我们永远不会要求你提供真实身份信息
2. **完全脱敏**：所有分享必须经过彻底脱敏
3. **社区审核**：所有分享都会经过社区审核，确保隐私安全
4. **随时删除**：如果你后悔分享，可以随时要求删除

## 分享奖励

虽然这不是一个商业项目，但我们会：
- 在生成的指南中标注贡献者（使用你指定的代号）
- 定期评选"最有价值分享"
- 建立贡献者荣誉榜

## 常见问题

**Q：我的经历很普通，值得分享吗？**

A：绝对值得！看似普通的经历，可能正是其他人正在面临的困境。你的分享可能帮助别人避免同样的错误。

**Q：我担心被同行认出怎么办？**

A：只要彻底脱敏，就不会被认出。我们建议你多检查几遍，确保没有任何可识别信息。

**Q：分享后如果客户看到怎么办？**

A：彻底脱敏后的分享不会暴露任何身份信息。如果你还是担心，可以等项目结束一段时间后再分享。

**Q：我可以分享成功的经验吗？**

A：当然可以！我们鼓励分享各种经验，包括成功的、失败的、以及正在进行的。
