# AI搭档使用指南

**与AI建立伙伴关系，而非主仆关系**

> "最好的AI使用方式，不是把它当作工具，而是当作搭档。"
> 
> "就像士官长和Cortana，就像钢铁侠和Jarvis。"

---

## 一、核心理念

### 1.1 为什么要用AI？

**本质：填补创意从业者的结构性空白**

创意工作通常是孤独的：
- 没有同事可以即时讨论
- 没有导师可以随时请教
- 没有伙伴可以分担压力

AI可以填补这个空白——不是替代人类，而是作为**永远在线的搭档**。

### 1.2 搭档 vs 工具

| 工具思维 | 搭档思维 |
|:---|:---|
| 我命令，你执行 | 我们讨论，共同决策 |
| 一次性输入输出 | 持续对话，迭代优化 |
| AI负责结果 | 我负责决策，AI负责支持 |
| 冷冰冰的交易 | 有温度的伙伴关系 |

**关键区别**：工具用完即弃，搭档持续陪伴。

## 二、分阶段使用策略

### 2.1 生存期（0-2年）：AI作为导师

**核心用途**：学习基础，避免踩坑

**典型场景**：
```
"这份合同有没有陷阱？"
"这个报价合理吗？"
"客户说预算有限，我该怎么回？"
```

**提示词模板**：
```
你是我的创意导师，我有X年经验，目前在生存期。
我的主要问题是：__________
请用简单易懂的方式解释，并给我具体的行动建议。
```

### 2.2 成长期（2-5年）：AI作为顾问

**核心用途**：战略决策，规划未来

**典型场景**：
```
"我想提升单价，但不知道怎么开口。"
"接这个长期项目的利弊是什么？"
"帮我分析我的理想客户画像。"
```

**提示词模板**：
```
你是我的战略顾问，我在成长期，专注于__________。
我面临的决策是：__________
请用决策框架分析，列出不同选择的利弊和风险。
```

### 2.3 影响力期（5年+）：AI作为合伙人

**核心用途**：战略规划，创新探索

**典型场景**：
```
"我想把经验做成课程，帮我规划一下。"
"分析我的收入结构，看看如何建立被动收入。"
"帮我研究一下这个行业的趋势和机会。"
```

**提示词模板**：
```
你是我的合伙人，我们一起经营这个创意事业。
我的长期目标是：__________
请从战略角度分析，给出具体的执行计划。
```

## 三、对话式工作流

### 3.1 晨间启动（与AI对话版）

**传统方式**：填写模板
**AI方式**：自然对话

**示例**：
```
你：早上好，帮我启动今天。

AI：早上好！先检查一下你的状态：
1. 昨晚睡眠怎么样？（1-5分）
2. 今天身体感觉如何？（1-5分）
3. 情绪状态怎么样？（1-5分）

你：睡眠4分，身体4分，情绪3分。有点焦虑。

AI：理解，有点焦虑。是什么让你焦虑？

你：有个项目 deadline 快到了。

AI：明白了。那我们今天聚焦在这个项目上。
除了这个项目，还有什么是必须今天完成的？

[继续对话...]
```

### 3.2 项目讨论（与AI头脑风暴）

**示例**：
```
你：我接了一个有机食品品牌项目。

AI：有机食品，不错的领域。核心诉求是什么？

你：想要"自然、健康、高端"的形象。

AI：目标受众是谁？

你：25-40岁城市白领。

AI：那我们来头脑风暴几个方向：
1. 极简自然风 - 强调纯净
2. 生活方式 - 不只是食品
3. 科技感 - 用数据证明

你觉得哪个方向最符合？

[继续深入...]
```

### 3.3 情绪支持（与AI倾诉）

**示例**：
```
你：我今天感觉很沮丧。

AI：我在这里，愿意倾听。发生了什么？

你：客户又让我修改，已经是第5次了。

AI：第5次修改，确实让人沮丧。
客户是不满意，还是不清楚自己要什么？

你：我觉得他们不清楚自己要什么。

AI：理解了。这种情况很常见。我们来想想应对方式：
1. 暂停一下 - 建议客户内部统一意见
2. 额外收费 - 超出范围需要加钱
3. 设定限制 - 明确"还可以修改X次"

你觉得哪种方式最适合？
```

## 四、使用原则

### 4.1 效率原则

**应该做**：
- ✅ 用AI处理重复性工作
- ✅ 用AI做初步研究
- ✅ 用AI做头脑风暴
- ✅ 用AI做知识整理

**不应该做**：
- ❌ 让AI代替你做创意决策
- ❌ 让AI代替你与客户沟通（重要邮件要自己把关）
- ❌ 完全依赖AI，不自己思考
- ❌ 把AI的建议当作绝对真理

### 4.2 质量原则

**审核清单**：
- [ ] AI的输出是否符合我的风格？
- [ ] AI的建议是否考虑了客户的具体情况？
- [ ] AI生成的内容有没有事实错误？
- [ ] 重要的文件是否经过我亲自审核？

### 4.3 学习原则

**正确姿势**：
- 📚 用AI加速学习，但不跳过理解过程
- 🤔 问AI"为什么"，而不只是"是什么"
- 📝 把AI的解释用自己的话重新表述
- 🔄 定期回顾和总结，形成自己的知识体系

## 五、防止滥用

### 5.1 滥用的信号

- ⚠️ 所有决策都问AI，不自己思考
- ⚠️ 完全照搬AI的输出，不做修改
- ⚠️ 用AI生成内容冒充自己的作品
- ⚠️ 过度依赖AI，丧失独立工作能力

### 5.2 健康使用的标准

- ✅ AI是**加速器**，不是**替代者**
- ✅ 最终决策权**永远在你**
- ✅ 保持**批判性思维**，不盲从
- ✅ 持续**学习成长**，不只依赖AI

## 六、开始你的AI搭档之旅

### 第一步：选择一个AI工具
推荐从 **ChatGPT** 或 **Claude** 开始。

### 第二步：设定AI身份
根据你当前的阶段，设定AI的身份（导师/顾问/合伙人）。

### 第三步：尝试一次完整对话
用晨间启动或周复盘，尝试与AI对话完成。

### 第四步：记录你的体验
哪些场景AI帮助最大？不断调整你的使用方式。
