# 阶段判断指南

**不确定自己在哪个阶段？用这份指南帮你判断**

> 阶段不是非黑即白的，你可能处于过渡期，需要同时看两个阶段的内容。

---

## 快速判断（30秒）

### 方法1：看核心指标

| 维度 | 生存期 | 成长期 | 影响力期 |
|:---|:---|:---|:---|
| **工作年限** | 0-2年 | 2-5年 | 5年+ |
| **月收入稳定性** | 不稳定，时有时无 | 基本稳定，有波动 | 稳定，可预测 |
| **客户来源** | 主要靠平台/偶然获得 | 50%+来自主动开发或转介绍 | 70%+来自品牌效应 |
| **单价水平** | 低于市场平均水平 | 市场平均水平或略高 | 高于市场平均水平 |
| **应急储备** | 少于3个月 | 3-6个月 | 6个月+ |

**判断规则**：
- 如果大多数指标符合某个阶段 → 你属于该阶段
- 如果指标分散在不同阶段 → 你处于过渡期
- 如果生存期指标仍有红色项 → 即使工作3年+，仍需补生存期内容

---

### 方法2：看当前最大痛点

**生存期典型痛点**：
- [ ] 这个月不知道下个项目在哪
- [ ] 客户报价总是压得很低
- [ ] 合同条款看不懂，担心被坑
- [ ] 工作时间长但收入低
- [ ] 经常要免费试稿、比稿

**成长期典型痛点**：
- [ ] 有客户但都是低价客户
- [ ] 想提价但不知道怎么开口
- [ ] 客户都是朋友介绍，没有主动获客能力
- [ ] 工作很忙但没有个人品牌
- [ ] 技能不错但市场认知度低

**影响力期典型痛点**：
- [ ] 收入全靠时间换钱，没有被动收入
- [ ] 想建立产品但不知道怎么开始
- [ ] 工作占据全部生活，没有自由
- [ ] 想影响行业但不知道如何开始
- [ ] 财务自由还有距离

**判断规则**：
- 勾选最多的类别 → 你目前最需要关注该阶段
- 如果多类都勾选了 → 从最基础的生存期开始

## 详细评估（3分钟）

### 生存期评估清单

**收入保障**（建议全部达标后再进入成长期）：
- [ ] 连续6个月有稳定收入
- [ ] 月收入覆盖基本生活开支
- [ ] 有3个月以上应急储备

**基础能力**（建议全部掌握）：
- [ ] 能看懂合同基本条款
- [ ] 知道如何报价和谈判
- [ ] 有标准的工作流程
- [ ] 能识别和拒绝不合理要求

**如果以上有任何一项未达标 → 建议继续巩固生存期基础，不要急于进入成长期**

> 💡 **提示**：这些指标是基于大量创意从业者经验总结的最佳实践，但每个人的情况不同。如果你某项差一点点但其他都很好，也可以尝试进入下一阶段，但要保持警惕。

---

### 成长期评估清单

**客户质量**（至少2项达标）：
- [ ] 50%+客户来自主动开发或转介绍
- [ ] 有3个以上长期合作客户
- [ ] 客户质量逐年提升
- [ ] 有拒绝客户的底气

**价值提升**（至少2项达标）：
- [ ] 过去1年单价有提升
- [ ] 有明确的差异化定位
- [ ] 有个人品牌积累（作品集、社交媒体等）
- [ ] 技能持续更新

**系统能力**（至少2项达标）：
- [ ] 有客户管理系统
- [ ] 有项目管理系统
- [ ] 有稳定的复盘习惯
- [ ] 有长期职业规划

**如果以上大部分达标 → 你进入成长期**

---

### 影响力期评估清单

**财务基础**（必须全部达标）：
- [ ] 年收入达到理想水平的80%+
- [ ] 有被动收入来源（版税、产品、投资等）
- [ ] 应急储备12个月+
- [ ] 有养老和保险规划

**影响力基础**（至少3项达标）：
- [ ] 在行业内有知名度
- [ ] 有分享经验的习惯
- [ ] 指导过新人或做过分享
- [ ] 参与过行业标准或社区建设

**自由基础**（至少2项达标）：
- [ ] 可以自主选择项目
- [ ] 可以控制工作时间和地点
- [ ] 工作不是生活的全部
- [ ] 有明确的理想生活图景

**如果以上大部分达标 → 你进入影响力期**

## 过渡期指导

### 什么是过渡期？

你处于过渡期，如果：
- 工作年限符合A阶段，但能力还在B阶段
- 部分指标符合A阶段，部分符合B阶段
- 正在从A阶段向B阶段努力

### 过渡期学习策略

**策略1：补齐短板**
- 先确保生存期所有基础达标
- 再追求成长期的目标
- 基础不牢，地动山摇

**策略2：交叉学习**
- 70%精力学习当前阶段内容
- 30%精力预习下阶段内容
- 例如：生存期用户70%看生存期，30%看成长期

**策略3：重点突破**
- 找出当前最大的1-3个痛点
- 集中资源解决这些痛点
- 不要试图同时解决所有问题

## 常见误区

### 误区1：只看工作年限
**错误**：工作3年就认为自己自动进入成长期
**正确**：看实际能力和状态，有人工作5年还在生存期

### 误区2：忽视生存期基础
**错误**：急着看成长期内容，忽视合同、报价等基础
**正确**：生存期基础不牢，成长期无法建立

### 误区3：阶段跳跃
**错误**：生存期还没稳定就想建立被动收入
**正确**：按阶段发展，不要跳步

### 误区4：阶段固化
**错误**：认为自己"就是生存期的人"
**正确**：阶段是动态变化的，通过努力可以进阶

## 行动建议

### 如果你确定了自己的阶段

1. **进入对应阶段目录**：[生存期](../01-生存期/) / [成长期](../02-成长期/) / [影响力期](../03-影响力期/)
2. **先看README**：了解该阶段的核心目标和内容结构
3. **找到当前最痛的点**：使用本指南的"当前最大痛点"部分
4. **开始阅读**：从最紧急的内容开始

### 如果你处于过渡期

1. **评估短板**：使用详细评估清单，找出未达标的项
2. **制定计划**：列出需要补齐的短板，按优先级排序
3. **交叉学习**：70%精力当前阶段，30%精力下阶段
4. **定期复评**：每月重新评估，跟踪进度

### 如果你还是不确定

1. **从生存期开始**：生存期内容对所有人都有价值
2. **使用AI搭档**：询问AI "我在哪个阶段？"
3. **寻求帮助**：在社区提问，让有经验的人帮你判断

## 阶段进阶路径

```
生存期 → 过渡期 → 成长期 → 过渡期 → 影响力期
  ↓         ↓         ↓         ↓          ↓
活下来   补齐短板   活得好   建立系统   活得自由
```

**关键提醒**：
- 进阶不是线性的，可能有反复
- 过渡期是正常的，不要焦虑
- 基础比速度更重要
- 定期评估，及时调整
