## 🏷️ 同义词对照表

同一个问题，可能有不同的描述方式：

| 你说的问题 | 对应文档关键词 |
|:---|:---|
| 客户不给钱 | 拖欠款项、拖延付款、欠款 |
| 客户要便宜点 | 降价、压价、砍价、预算不够 |
| 客户要试稿 | 免费试稿、测试、样稿 |
| 客户要改来改去 | 修改、变更、范围蔓延 |
| 做不完了 | deadline、延期、超时 |
| 太累了 | burnout、倦怠、疲惫、不想工作 |
| 不知道收多少钱 | 报价、定价、收费、预算 |
| 找不到活 | 客户开发、获客、接单、找项目 |
| 想多赚钱 | 涨价、提升单价、收入增长 |
| 想让别人知道我 | 品牌建设、知名度、影响力 |

## 🎯 使用建议

### 方法1：直接浏览

如果你大致知道问题类型，直接浏览上方的分类：
- 客户相关问题
- 时间和项目管理问题
- 定价和报价问题
- ...

### 方法2：关键词搜索

使用浏览器的搜索功能（`Ctrl+F` 或 `Cmd+F`），搜索你的问题关键词：
1. 按 `Ctrl+F`
2. 输入你的问题关键词（如"拖欠"、"报价"）
3. 查看匹配结果

### 方法3：查看应急手册

如果是**紧急情况**，直接查看 [应急手册](应急手册.md)：
- 列出了最常见的15种紧急情况
- 每种情况都有立即可以使用的解决方案

### 方法4：使用AI评估

如果不确定自己的问题属于哪一类，使用 [对话式职业评估](对话式职业评估.md)：
- AI会帮你分析问题
- 给出针对性的建议
- 推荐相关工具

## 🆘 还是找不到？

**如果按问题查找还是找不到你需要的内容**：

1. **尝试不同关键词**
   - 用同义词对照表找其他说法
   - 用更笼统或更具体的词

2. **查看工具库索引**
   - [工具库索引](工具库索引.md) 按场景分类
   - 可能你的问题属于某个场景

3. **使用AI搜索**
   - 把问题描述给AI（ChatGPT/Claude）
   - 问"CCOS中如何解决[你的问题]"

4. **在社区求助**
   - [查看社区与支持](社区与支持.md)
   - 在GitHub Issues中提问
   - 描述你的问题，社区会帮你找到答案
