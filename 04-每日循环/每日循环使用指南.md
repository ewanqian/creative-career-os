# 每日循环使用指南

**用15分钟，让每一天都高效**

> 不是时间管理，而是能量管理。

---

## ⚡ 超快速了解（30秒）

**每日循环是什么？**
- 每天15分钟的自我管理系统
- 晨间5分钟：启动高效一天
- 晚间10分钟：复盘今日，规划明日

**为什么要用？**
- ✅ 从"被推着走"到"主动掌控"
- ✅ 从"瞎忙"到"有重点"
- ✅ 从"焦虑"到"平静"

**适合谁？**
- 工作没有规律，经常拖延
- 每天很忙但效率不高
- 想建立工作习惯，但坚持不下来

---

## 🌅 晨间启动（5分钟）

### 什么时候用？

**最佳时间**：起床后或开始工作前
**必须完成**：在开始工作前完成

### 极简版（1分钟）

如果你很忙，只回答这3个问题：

1. **今天最重要的1件事是什么？**
2. **今天需要避免什么干扰？**
3. **今天什么时候休息？**

### 完整版（5分钟）

使用 [晨间启动模板](晨间启动模板.md)

**填写顺序**：
1. 状态检查（1分钟）——了解今天的状态
2. 今日聚焦（2分钟）——确定今天的重点
3. 时间块规划（2分钟）——安排今天的时间

### 使用示例

**场景**：自由设计师，今天有一个项目 deadline

```
【晨间启动】2024年2月6日 周二

1. 状态检查
   - 睡眠：7小时，质量良好
   - 身体：无不适
   - 情绪：有点焦虑（因为 deadline）
   - 能量：7/10

2. 今日聚焦
   - 最重要1件事：完成XX项目的最终交付
   - 其他任务：回复客户邮件、准备明天的会议
   - 今日不做什么：不接新项目、不刷社交媒体

3. 时间块规划
   - 9:00-12:00：专注完成项目（关闭微信）
   - 12:00-13:00：午餐+休息
   - 13:00-15:00：修改优化+测试
   - 15:00-16:00：交付+写邮件
   - 16:00-17:00：回复其他客户邮件
   - 17:00-18:00：准备明天会议
```

## 🌙 晚间复盘（10分钟）

### 什么时候用？

**最佳时间**：工作结束后或睡前
**必须完成**：在当天结束前完成

### 极简版（2分钟）

如果你很累，只回答这3个问题：

1. **今天完成了什么？**
2. **今天学到了什么？**
3. **明天最重要的事是什么？**

### 完整版（10分钟）

使用 [晚间复盘模板](晚间复盘模板.md)

**填写顺序**：
1. 今日回顾（3分钟）——回顾今天做了什么
2. 深度反思（4分钟）——思考今天的问题和收获
3. 明日准备（3分钟）——为明天做准备

### 使用示例

**场景**：同上

```
【晚间复盘】2024年2月6日 周二

1. 今日回顾
   - 完成任务：
     ✅ 完成XX项目最终交付（比计划提前1小时）
     ✅ 回复3个客户邮件
     ✅ 准备明天会议资料
   - 未完成：
     ❌ 学习新软件（没时间）

2. 深度反思
   - 做得好的：
     - 早上专注时间效率高，关闭了微信
     - 项目提前完成，说明时间预估准确
   - 可以改进的：
     - 下午有点累，效率下降，应该安排午休
     - 回复邮件花了太多时间，应该集中处理
   - 学到的：
     - 关闭社交软件确实能提高效率
     - deadline 前1天完成，留出缓冲时间很重要

3. 明日准备
   - 明天最重要的事：参加XX客户会议
   - 需要准备的资料：作品集、报价单
   - 明天要注意的：提前15分钟到，检查设备
```

## 🎯 21天养成计划

### 为什么21天？

研究表明，养成一个新习惯需要21天。每日循环是一个习惯，需要时间来建立。

### 第1周：建立习惯（第1-7天）

**目标**：每天完成晨间启动，不管内容多少

**策略**：
- 使用极简版（1-2分钟）
- 设定固定时间（如每天早上9点）
- 设置手机提醒
- 允许不完美，先完成再完美

**检查点**：
- [ ] 连续7天完成晨间启动
- [ ] 找到适合自己的时间

### 第2周：增加晚间复盘（第8-14天）

**目标**：每天完成晨间+晚间

**策略**：
- 晨间：极简版（1分钟）
- 晚间：极简版（2分钟）
- 周末可以使用完整版

**检查点**：
- [ ] 连续7天完成晨间+晚间
- [ ] 开始感受到效果（更有掌控感）

### 第3周：优化和坚持（第15-21天）

**目标**：找到适合自己的节奏

**策略**：
- 工作日：极简版
- 周末：完整版
- 根据情况调整问题

**检查点**：
- [ ] 连续21天完成
- [ ] 形成习惯，不需要提醒

### 21天后

**评估效果**：
- 工作效率是否提高？
- 焦虑感是否减少？
- 是否更有掌控感？

**决定下一步**：
- 如果有效 → 继续使用，可以尝试完整版
- 如果无效 → 调整问题，或者尝试其他方法

## 💡 使用技巧

### 技巧1：绑定现有习惯

**方法**：把每日循环绑定到已有的习惯上

**示例**：
- 晨间：喝咖啡时填写晨间启动
- 晚间：刷牙前填写晚间复盘

**好处**：利用现有习惯的惯性，更容易坚持

### 技巧2：降低门槛

**方法**：从极简版开始，不要追求完美

**示例**：
- 第1周：只写1句话
- 第2周：写3句话
- 第3周：尝试完整版

**好处**：降低心理压力，更容易开始

### 技巧3：可视化进度

**方法**：用日历或打卡表记录完成情况

**示例**：
```
2月每日循环打卡
日 一 二 三 四 五 六
             ✅ ✅ ✅
✅ ✅ ✅ ✅ ✅ ✅ ✅
...
```

**好处**：视觉反馈，增加成就感

### 技巧4：找到 accountability partner（问责伙伴）

**方法**：找一个朋友，互相监督

**示例**：
- 每天互相发送晨间启动截图
- 每周互相检查完成情况

**好处**：外部监督，增加坚持的动力

### 技巧5：定期回顾效果

**方法**：每周或每月回顾每日循环的效果

**示例**：
- 每周日晚回顾本周的晨间启动和晚间复盘
- 问自己：这个方法对我有帮助吗？
- 如果有，继续；如果没有，调整

**好处**：确保方法有效，避免无效坚持

## ⚠️ 常见问题

### Q1：某天太忙，没时间做怎么办？

**A**：使用极简版，1-2分钟完成。或者，至少回答这1个问题：
- 晨间：今天最重要的事是什么？
- 晚间：今天完成了什么？

**原则**：做一点比不做好，不要完美主义。

### Q2：感觉填写很重复，没有新鲜感怎么办？

**A**：
1. 调整问题，根据自己的情况定制
2. 每周或每月更换问题
3. 使用AI搭档，用对话代替填写

### Q3：坚持了一段时间，但感觉没有效果怎么办？

**A**：
1. 检查是否认真填写，还是敷衍了事
2. 回顾填写的内容，是否有行动改变
3. 尝试调整问题，找到更适合自己的
4. 如果确实无效，尝试其他方法（如周复盘）

### Q6：如果中断了怎么办？

**A**：

**第1步：不要自责**
- 中断是正常的，不要因此放弃
- 告诉自己："明天重新开始"

**第2步：分析原因**
- 为什么中断？（太忙？忘记？觉得没用？）
- 找到原因，才能解决问题

**第3步：调整策略**
- 如果太忙 → 使用极简版
- 如果忘记 → 设置更强的提醒
- 如果觉得没用 → 调整问题或尝试其他方法

**第4步：重新开始**
- 不需要从第1天开始，继续计数
- 例如：中断了3天，之前坚持了10天，现在继续第11天
- 重点是持续，不是完美

**关键原则**：
- 中断1天 ≠ 失败
- 中断1周 ≠ 失败
- 中断1个月 ≠ 失败
- **放弃** = 失败

**记住**：
> "重要的不是从不中断，而是中断后重新开始。"
> 
> "完美的坚持不如不完美的持续。"

### Q4：晨间启动和晚间复盘都要做吗？

**A**：
- 理想情况：两个都做
- 最低要求：至少做晨间启动
- 如果只能选一个：选晨间启动（更有前瞻性）

### Q5：可以只用AI搭档，不用模板吗？

**A**：可以！CCOS 2.0 支持AI对话式每日循环：
- 对AI说"帮我启动今天"
- AI会引导你完成晨间启动
- 对AI说"帮我复盘今天"
- AI会引导你完成晚间复盘

**好处**：更自然，像和朋友聊天
