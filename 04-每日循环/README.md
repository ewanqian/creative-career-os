## ⚡ 极简快速开始（1分钟）

**如果你现在就想开始，只回答这3个问题**：

**晨间（1分钟）**：
1. 今天最重要的1件事是什么？
2. 今天需要避免什么干扰？
3. 今天什么时候休息？

**晚间（2分钟）**：
1. 今天完成了什么？
2. 今天学到了什么？
3. 明天最重要的事是什么？

**完成！** 🎉

## 🎯 建议使用流程

### 第1天：阅读指南
- [ ] 阅读 [每日循环使用指南](每日循环使用指南.md)
- [ ] 决定使用极简版还是完整版
- [ ] 设定固定时间（如早上9点晨间，晚上10点晚间）

### 第1周：建立习惯
- [ ] 每天完成晨间启动（极简版）
- [ ] 设置手机提醒
- [ ] 允许不完美，先完成再完美

### 第2周：增加晚间复盘
- [ ] 每天完成晨间+晚间（极简版）
- [ ] 周末可以尝试完整版

### 第3周：优化和坚持
- [ ] 找到适合自己的节奏
- [ ] 根据情况调整问题

## 💡 替代方案：AI对话式

**不想填写模板？用AI搭档！**

**晨间**：对AI说"帮我启动今天"
- AI会问你3-5个问题
- 自然对话，像和朋友聊天

**晚间**：对AI说"帮我复盘今天"
- AI会引导你回顾今天
- 总结收获，规划明天

[查看AI搭档使用指南 →](../docs/AI搭档使用指南.md)

## ⚠️ 遇到困难？

| 问题 | 解决方案 |
|:---|:---|
| 某天太忙没时间 | 使用极简版，1-2分钟完成 |
| 感觉重复没新鲜感 | 调整问题，或使用AI对话式 |
| 坚持不下来 | 找问责伙伴互相监督 |
| 感觉没效果 | 检查是否认真填写，或调整问题 |
