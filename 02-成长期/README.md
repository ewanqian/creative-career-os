## 💡 成长期进阶法则

### ✅ 应该做的
- **提升单价**：每年至少提升10-20%
- **建立品牌**：持续输出内容，建立专业形象
- **主动获客**：不再被动等待，主动寻找理想客户
- **维护关系**：把一次性客户变成长期合作伙伴
- **持续学习**：每季度学习一个新技能

### ❌ 不应该做的
- 不要**贱卖自己**（成长了就要涨价）
- 不要**接所有项目**（学会筛选客户）
- 不要**单打独斗**（建立同行网络）
- 不要**忽视健康**（ burnout 风险增加）
- 不要**停止学习**（行业在变化）

## 🤖 AI搭档策略（成长期）

在成长期，把AI当作**顾问**——一个战略顾问，帮你做决策、规划未来。

### 成长期AI使用场景

**1. 定价策略**
```
"我想提升单价，但不知道怎么跟老客户开口。"
"分析我的服务，看看哪些地方可以溢价。"
```

**2. 品牌建设**
```
"我想建立个人品牌，从哪开始？"
"帮我规划一个3个月的内容营销计划。"
```

**3. 客户开发**
```
"我想主动找客户，有什么策略？"
"帮我分析我的理想客户画像。"
```

**4. 决策支持**
```
"我在考虑要不要接一个长期项目，利弊是什么？"
"用决策框架分析这个选择。"
```
