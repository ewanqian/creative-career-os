## 🎯 这到底是什么？

### 一句话解释
**创意行业的"维基百科+工具箱"** - 不是一个人写的攻略，而是所有人共同编写的生存手册。

### 为什么需要这个？
创意行业有个残酷的现实：
- ❌ **学校教设计，不教怎么签合同**
- ❌ **公司教技能，不教怎么谈价格**
- ❌ **这些"暗知识"只能靠踩坑获得，或者靠私人关系传递**

**结果**：新人成为系统性风险的主要承担者，老人把经验带进坟墓。

### 我们的解法
**把分散的、隐性的生存知识，变成结构化的、可复用的公共资产。**

就像：
- 程序员有 **Stack Overflow**
- 医生有 **医学指南**
- 创意从业者有 **Creative-Career-OS**
