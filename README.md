## ✨ 5分钟了解 CCOS

**CCOS 是什么？**

CCOS（Creative Career Operating System，创意职业操作系统）是一套帮助创意从业者（设计师、插画师、摄影师、写作者等）从生存到影响力完整职业生涯的导航系统。

**为什么需要 CCOS？**

作为创意从业者，你经常面临：
- 😰 客户要免费试稿，不知道怎么拒绝
- 💰 不知道怎么报价，怕报高了吓跑客户，报低了自己吃亏
- 📄 合同条款看不懂，担心被坑
- 😫 工作不稳定，收入忽高忽低
- 🤔 想提升单价，但不知道怎么开口
- 😵‍💫 每天忙碌但感觉没有成长

**CCOS 如何解决这些问题？**

| 你的问题 | CCOS 的解决方案 |
|:---|:---|
| 客户要免费试稿 | [应对话术模板](01-生存期/基础保护/沟通-应对免费试稿-v1.0.md) |
| 不知道怎么报价 | [报价策略指南](01-生存期/基础保护/报价-应对无预算试探-v1.0.md) + [报价单模板](tools/实用模板/项目报价单模板.md) |
| 合同怕有坑 | [合同审查清单](01-生存期/基础保护/合同-著作权归属界定-v1.0.md) + [合同模板](tools/实用模板/项目合同模板.md) |
| 收入不稳定 | [客户开发策略](02-成长期/客户开发/主动-客户开发策略-v1.0.md) + [被动收入构建](03-影响力期/规模化/进攻-被动收入构建-v1.0.md) |
| 想提升单价 | [价值定价策略](02-成长期/价值提升/主动-价值定价策略-v1.0.md) |
| 感觉没有成长 | [每日/每周/每月复盘系统](04-每日循环/) |

**CCOS 的核心理念**

> "从独自探索到系统支持，从迷茫试错到有迹可循"

我们不给你空洞的理论，而是提供：
- ✅ **立即可用的模板**（报价单、合同、邮件话术）
- ✅ **经过验证的策略**（来自真实从业者的经验）
- ✅ **系统性的框架**（从生存到影响力的完整路径）
- ✅ **AI 搭档支持**（让 AI 成为你的导师/顾问/合伙人）
