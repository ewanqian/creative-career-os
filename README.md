## 🔄 循环系统

### 每日循环（Daily Loop）

**晨间启动**（15分钟）
- [晨间启动模板](04-每日循环/晨间启动模板.md)
- 状态检查 → 今日聚焦 → 时间块规划

**工作推进**
- 深度工作法（90分钟专注块）
- 番茄工作法（25+5循环）
- 状态调整技巧

**晚间复盘**（15分钟）
- [晚间复盘模板](04-每日循环/晚间复盘模板.md)
- 今日回顾 → 明日规划 → 感恩记录

---

### 每周检查（Weekly Review）

**用30分钟回顾本周，规划下周**

- [周复盘模板](05-每周检查/周复盘模板.md)
- 项目盘点：完成、风险、安排
- 客户维护：沟通、跟进、转介绍
- 自我关怀：时长、休息、状态

---

### 每月复盘（Monthly Review）

**用1小时深度复盘本月，规划下月**

- [月复盘模板](06-每月复盘/月复盘模板.md)
- 财务复盘：收入、支出、储蓄
- 职业发展：学习、作品、网络
- 生活平衡：健康、关系、意义
- 状态判定：生存/成长/影响力期
