# Creative Career OS

**为你的创意生涯，提供全方位的支持。**

Creative Career OS 是一套由创意从业者共同构建的资源系统。我们将每个人在职业道路上积累的经验，转化为你可以立即使用的指南、工具和最佳实践。

无论你是刚刚起步，还是希望突破瓶颈，这里都有适合你的资源。

---

## 三大核心体验

### 🛡️ 保护

在复杂的商业环境中，保护你的权益和创作成果。

了解如何：
- 签订公平的合同
- 合理定价你的服务
- 有效沟通，避免误解

[探索保护资源](#保护)

### 🌱 成长

从执行者成长为有影响力的创意专业人士。

学习如何：
- 建立个人品牌
- 找到你的独特定位
- 拓展收入来源

[探索成长资源](#成长)

### 🧠 滋养

保持身心健康，实现可持续的职业发展。

掌握如何：
- 管理工作压力
- 建立健康的工作节奏
- 保持创造动力

[探索滋养资源](#滋养)

## 三种使用方式

### 方式1：直接阅读
打开任意资源，直接阅读Markdown文档，复制需要的内容使用。

### 方式2：下载离线使用
点击右上角 **Code** → **Download ZIP**，下载完整资源库，离线使用。

### 方式3：参与共建
阅读 [如何贡献](docs/01-经验分享指南.md)，分享你的经验，帮助更多人。
