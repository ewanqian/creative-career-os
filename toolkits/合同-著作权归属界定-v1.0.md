# 著作权归属界定条款模组 v1.0

## 元信息
- **来源病例**：#001（客户以"比稿"名义征集方案后融合创意）
- **共识形成日期**：2026-02-02
- **最后更新**：2026-02-02
- **贡献者**：Creative-Career-OS 社区
- **适用场景**：品牌设计、视觉设计、插画创作、UI设计等创意工作

## 核心策略
通过**分级授权**和**阶段确权**机制，在保护创作者权益的同时满足客户使用需求，避免"版权全部归属甲方"的陷阱。

## 详细内容

### 方案A：分级授权模式（推荐）

**核心思路**：将版权拆分为不同层级，客户按需求购买相应权限，而非一次性买断所有权利。

**条款模板**：
```
第X条 知识产权归属与授权

1. 作品著作权归属
   本合同项下创作的作品（以下简称"作品"）的完整著作权归乙方（创作者）所有。

2. 授权范围与期限
   甲方（客户）获得以下非独占使用权：
   
   a) 基础使用权：
      - 使用范围：[具体使用场景，如品牌宣传、产品包装、线上推广]
      - 使用期限：自交付之日起[X年/永久]
      - 使用地域：[中国大陆/全球]
   
   b) 扩展使用权（可选，需另行协商费用）：
      - 衍生开发权：基于作品开发新作品的权利
      - 转授权：允许第三方使用的权利
      - 独占使用权：排除乙方自身使用的权利

3. 署名权保留
   乙方保留在作品上署名的权利，甲方在使用作品时应保留乙方署名，除非双方另有书面约定。

4. 作品交付与确权
   - 初稿阶段：乙方保留全部权利，甲方仅有审阅权
   - 终稿确认：甲方支付尾款后，获得约定的使用权
   - 源文件：源文件所有权归乙方，甲方可获得约定格式的使用文件

5. 违约责任
   任何一方超出约定范围使用作品的，应承担违约责任，赔偿对方损失。
```

### 方案B：阶段确权模式

**核心思路**：根据项目阶段逐步转移权利，而非在项目开始时即确定全部归属。

**条款模板**：
```
第X条 分阶段知识产权安排

1. 概念阶段（初稿）
   - 著作权：完全归乙方所有
   - 甲方权利：仅可审阅并提出修改意见
   - 如项目终止：甲方不得使用任何初稿内容

2. 执行阶段（修改稿）
   - 著作权：仍归乙方所有
   - 甲方权利：可内部讨论、向上汇报
   - 限制：不得对外展示、不得用于商业用途

3. 完成阶段（终稿）
   - 甲方支付全部费用后，获得【具体约定的使用权】
   - 乙方保留【署名权/作品集使用权/案例展示权】

4. 特别约定
   - 比稿保护：如采用比稿形式，未中标方案著作权完全归乙方，甲方不得使用
   - 修改限制：超出约定次数的修改，著作权归属需重新协商
```

## 使用指南

### 何时使用
- ✅ 品牌设计项目（VI、logo、包装）
- ✅ 视觉创作项目（插画、海报、主视觉）
- ✅ UI/UX设计项目
- ✅ 任何涉及原创创意的委托创作

### 何时不适用
- ❌ 雇佣关系下的职务作品（需遵循劳动法相关规定）
- ❌ 法律法规明确规定著作权归特定主体的情形
- ❌ 客户明确要求且合理支付买断费用的特殊项目

### 注意事项
1. **明确界定"作品"范围**：在合同中清晰定义交付物包括哪些内容
2. **细化使用场景**：尽可能详细描述客户的使用场景，避免模糊表述
3. **保留证据**：保存创作过程文件、沟通记录，以备维权需要
4. **定期审查**：随着项目进展，及时确认权利归属状态
5. **法律咨询**：重要项目建议咨询专业知识产权律师

## 话术模板

### 面对"版权全部归甲方"要求时

**初级回应**（温和但坚定）：
```
感谢您的信任。关于版权归属，我建议采用分级授权的方式。

这样既能满足贵司的使用需求，也能保护我们作为创作者的基本权益。

具体而言，贵司可以获得在[具体场景]的完整使用权，而我保留署名权和在作品集中展示的权利。

这样既符合行业惯例，也能确保双方的权益都得到保障。您看这样的安排是否可以？
```

**进阶回应**（专业且有理有据）：
```
理解贵司希望获得完整使用权的考虑。不过，从专业角度建议采用"授权使用"而非"买断版权"的模式。

原因如下：
1. 成本控制：买断版权通常需要支付3-5倍的费用，而授权使用更经济
2. 灵活性：您可以根据实际需要选择授权范围和期限
3. 后续服务：保留版权有利于我们提供后续的延展设计和更新服务

我建议的授权方案是：贵司获得[具体范围]的永久使用权，费用为[金额]，这样既满足需求，又合理控制成本。
```

**底线回应**（对方坚持时的谈判策略）：
```
如果贵司确实需要买断版权，我需要调整报价。因为这意味着：
1. 我无法再使用该作品作为案例
2. 失去了后续授权给其他客户的潜在收益
3. 需要签署更严格的保密和竞业限制

基于以上考虑，买断版权的报价为[原报价的3-5倍]。

或者，我们可以折中：贵司获得独占使用权（排除我授权给竞品），但保留我的署名权和案例展示权，费用为[原报价的1.5-2倍]。
```
