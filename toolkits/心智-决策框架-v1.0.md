## 核心概念

### 为什么创意从业者需要决策框架？

**创意工作的决策特点**：
- 不确定性高：结果难以预测
- 影响因素多：创意、商业、个人喜好
- 信息不完整：市场变化快，难以收集全部信息
- 情感因素重：创意工作往往与个人情感绑定

**决策困难的原因**：
- 信息过载，难以判断
- 完美主义倾向，追求最优解
- 害怕承担责任和后果
- 缺乏系统的决策方法

### 决策的本质

**不是**：追求完美无缺的选择
**是**：在有限信息下，做出当下最优的选择

**核心公式**：
```
好决策 = 明确目标 + 充分信息 + 系统分析 + 勇气行动
```

| 决策类型 | 不确定性 | 方法 | 示例 |
|:---|:---:|:---|:---|

| **不确定型** | 高 | 情景规划 | 结果高度不确定的选择 |

## 通用决策框架

### 框架1：RICE 决策法

**适用场景**：项目优先级、任务排序

**步骤**：
1. **R (Reach)**：影响范围
   - 影响多少人？
   - 覆盖多少场景？

2. **I (Impact)**：影响程度
   - 影响有多深？
   - 1-5分评分

3. **C (Confidence)**：信心程度
   - 对结果的信心？
   - 1-100%评分

4. **E (Effort)**：投入精力
   - 需要多少时间和资源？
   - 1-5分评分

**计算**：
```
RICE分数 = (Reach × Impact × Confidence) ÷ Effort
```

**示例**：
```
项目A：
- Reach: 100人
- Impact: 4分
- Confidence: 80%
- Effort: 3分
RICE = (100×4×0.8)÷3 = 106.7

项目B：
- Reach: 50人
- Impact: 5分
- Confidence: 90%
- Effort: 2分
RICE = (50×5×0.9)÷2 = 112.5

选择：项目B
```

### 框架2： pros & cons 增强版

**适用场景**：二元选择（是/否、A/B选择）

**步骤**：
1. **列出所有因素**
   - 优势（Pros）
   - 劣势（Cons）
   - 机会（Opportunities）
   - 威胁（Threats）

2. **加权评分**
   - 给每个因素重要性评分（1-5分）
   - 给每个因素实现可能性评分（1-10分）

3. **计算总分**
   - 正因素：重要性 × 可能性
   - 负因素：-（重要性 × 可能性）

**示例**：
```
是否接下低预算但有名气的项目

Pros：
- 提升作品集质量（重要性5×可能性9=45）
- 增加行业曝光（重要性5×可能性8=40）
- 可能带来后续合作（重要性4×可能性6=24）

Cons：
- 时薪低（重要性4×可能性10=-40）
- 占用其他项目时间（重要性3×可能性8=-24）
- 可能影响定价体系（重要性3×可能性6=-18）

总分：45+40+24-40-24-18=27

结论：可以接
```

### 框架3：10/10/10 决策法

**适用场景**：重大人生和职业决策

**步骤**：
1. **10分钟后**：我会如何看待这个决定？
2. **10个月后**：我会如何看待这个决定？
3. **10年后**：我会如何看待这个决定？

**示例**：
```
是否接受一份高薪但不喜欢的工作

10分钟后：开心（高薪）
10个月后：痛苦（不喜欢）
10年后：后悔（浪费了时间在不喜欢的事情上）

结论：不接受
```

---

### 框架4：最小可行测试（MVP）

**适用场景**：高不确定性的创新决策

**步骤**：
1. **设定假设**：如果我做X，会得到Y结果
2. **设计测试**：用最小成本验证假设
3. **执行测试**：收集数据和反馈
4. **分析结果**：根据数据做决定

**示例**：
```
是否开设计课程

假设：如果我开设计课程，会有足够的人报名

测试：
- 写一篇课程大纲文章
- 收集预报名
- 设定最低人数阈值

结果：
- 30人预报名，超过20人阈值

结论：可以开
```

## 职业决策专项框架

### 场景1：是否接新项目

**决策树**：
```
Step 1: 基本条件检查
□ 时间是否冲突？
□ 技能是否匹配？
□ 预算是否合理？

Step 2: 价值评估
□ 作品集价值（1-5）
□ 经验价值（1-5）
□ 关系价值（1-5）
□ 财务价值（1-5）

Step 3: 风险评估
□ 客户风险（1-5）
□ 质量风险（1-5）
□ 时间风险（1-5）

Step 4: 综合判断
总价值 > 总风险 + 机会成本？
```

### 场景2：职业转型

**决策矩阵**：

| 因素 | 评分 | 权重 | 加权分 |
|:---|:---:|:---:|:---:|
| **个人兴趣** | 1-5 | 0.3 | |
| **技能匹配** | 1-5 | 0.25 | |
| **市场需求** | 1-5 | 0.2 | |
| **收入潜力** | 1-5 | 0.15 | |
| **工作生活平衡** | 1-5 | 0.1 | |

**总分**：加权分总和
**阈值**：≥3.5分可以考虑转型

### 场景3：客户选择

**筛选标准**：
```
必选条件（必须满足）：
□ 预算合理
□ 尊重专业
□ 沟通顺畅

加分条件（越多越好）：
□ 行业匹配度高
□ 项目有意义
□ 长期合作潜力
□ 可以展示

减分条件（尽量避免）：
□ 要求免费试稿
□ 无限修改倾向
□ 付款拖延历史
□ 行业口碑差
```

## 决策陷阱识别

### 认知偏差

**1. 确认偏差**
- **症状**：只寻找支持自己已有观点的信息
- **应对**：主动寻找相反证据

**2. 沉没成本谬误**
- **症状**：因为已经投入，继续做不划算的事情
- **应对**：忽略过去的投入，只看未来

**3. 锚定效应**
- **症状**：过度依赖第一个接收到的信息
- **应对**：收集多个参考点

**4. 损失厌恶**
- **症状**：害怕损失超过对获得的渴望
- **应对**：客观评估风险和收益

**5. 群体思维**
- **症状**：为了融入群体而放弃独立思考
- **应对**：寻求不同意见

### 情绪干扰

**1. 决策疲劳**
- **症状**：做了太多决策后，判断力下降
- **应对**：重要决策放在精力充沛时

**2. 恐惧影响**
- **症状**：害怕失败，不敢做决定
- **应对**：想象最坏结果，制定应对方案

**3. 冲动决策**
- **症状**：情绪激动时快速做决定
- **应对**：设置冷静期

## 决策后管理

### 执行计划

**SMART 目标**：
- Specific（具体）
- Measurable（可衡量）
- Achievable（可实现）
- Relevant（相关）
- Time-bound（时间限制）

**示例**：
```
决策：接下品牌设计项目

执行计划：
- 第1周：完成需求调研和品牌策略
- 第2周：提交3个概念方案
- 第3周：根据反馈深化方案
- 第4周：完成最终交付
```

### 跟踪与调整

**定期回顾**：
- 设定检查点
- 收集数据和反馈
- 及时调整

**失败管理**：
- 接受失败是正常的
- 分析原因，总结教训
- 不要过度自责
