## 核心概念

### 为什么需要明确违约责任？

**没有违约责任条款的风险**：
- 客户拖延付款，无法追责
- 客户随意取消项目，自己白忙活
- 自己延期交付，被客户索赔
- 发生争议时，没有解决依据

**明确违约责任的好处**：
- 约束双方行为
- 降低违约概率
- 发生违约时有处理依据
- 保护双方合法权益

## 常见违约情形

### 客户违约

**1. 付款违约**
- 拖延付款
- 拒绝付款
- 要求不合理扣款

**2. 配合违约**
- 不提供必要资料
- 不按时反馈
- 决策人失联

**3. 范围违约**
- 要求超出合同范围的工作
- 无限次修改
- 擅自使用未验收作品

**4. 取消违约**
- 项目进行中单方面取消
- 无正当理由终止合作

### 设计师违约

**1. 交付违约**
- 延期交付
- 交付质量不达标
- 未按约定完成全部工作

**2. 保密违约**
- 泄露客户商业信息
- 提前公开未发布作品

**3. 版权违约**
- 使用未经授权的素材
- 侵犯第三方版权

## 违约责任条款模板

### 条款1：付款违约责任

```
第X条 付款违约责任

1. 甲方逾期付款
   甲方未按约定时间付款的，每逾期一日，应按未付款项的0.5%向乙方支付违约金。
   逾期超过15日的，乙方有权：
   a) 暂停工作直至收到款项
   b) 解除合同并要求甲方支付已完成工作的费用
   c) 保留已交付作品的使用权直至收到全部款项

2. 甲方拒绝付款
   甲方无正当理由拒绝付款的，乙方有权：
   a) 要求甲方支付全部合同款项
   b) 要求甲方支付违约金（合同总额的20%）
   c) 通过法律途径追讨
   d) 在行业内公示甲方违约行为（脱敏处理）
```

### 条款2：交付违约责任

```
第X条 交付违约责任

1. 乙方延期交付
   因乙方原因延期交付的：
   a) 延期3日内：乙方应尽快完成，不收取违约金
   b) 延期3-7日：乙方应支付合同总额1%的违约金/日
   c) 延期超过7日：甲方有权解除合同，乙方应退还已收款项

2. 免责情形
   以下情形导致的延期，乙方不承担责任：
   a) 甲方未按时提供必要资料
   b) 甲方反馈延迟超过约定时间
   c) 甲方要求重大修改导致工作量增加
   d) 不可抗力因素

3. 延期处理流程
   乙方预计无法按期交付时，应：
   a) 提前3日书面通知甲方
   b) 说明延期原因
   c) 提供新的交付时间
   d) 协商解决方案
```

### 条款3：项目取消责任

```
第X条 项目取消责任

1. 甲方取消项目
   甲方在项目进行过程中取消项目的，应按以下标准支付费用：
   a) 概念阶段取消：支付合同总额的30%
   b) 设计阶段取消：支付合同总额的60%
   c) 完稿阶段取消：支付合同总额的100%

2. 乙方取消项目
   乙方无正当理由取消项目的，应：
   a) 退还已收全部款项
   b) 支付合同总额20%的违约金
   c) 移交已完成的工作成果

3. 协商解除
   双方协商一致解除合同的，应：
   a) 根据已完成工作量结算
   b) 签署解除协议
   c) 明确后续权利义务
```

### 条款4：保密违约责任

```
第X条 保密违约责任

1. 甲方保密义务
   甲方不得将乙方提供的未公开设计方案泄露给第三方。
   违反保密义务的，应支付合同总额50%的违约金，
   并赔偿乙方因此遭受的全部损失。

2. 乙方保密义务
   乙方不得泄露甲方的商业信息、用户数据等机密信息。
   违反保密义务的，应支付合同总额50%的违约金，
   并赔偿甲方因此遭受的全部损失。
```

## 违约处理流程

### 当客户违约时

**Step 1：友好提醒**
```
[客户名称]您好，

根据合同约定，[款项/资料/反馈]应在[日期]前[支付/提供]。
目前已逾期[X]天。

请尽快处理，以免影响项目进度。
如有特殊情况，请及时沟通。

谢谢！
```

**Step 2：正式警告**
```
[客户名称]您好，

关于[违约事项]，我方已于[日期]提醒，但至今未解决。

根据合同第[X]条，您的行为已构成违约。
请在[3日内]解决，否则我方将：
- 暂停工作
- 按合同约定收取违约金
- 保留采取进一步法律措施的权利

请重视此事，及时沟通解决。
```

**Step 3：执行违约条款**
- 暂停工作
- 收取违约金
- 寻求法律途径

**Step 4：善后处理**
- 结算已完成工作
- 移交工作成果（如适用）
- 保留证据
- 总结经验

### 当自己可能违约时

**Step 1：提前预警**
发现可能无法按期交付时，立即通知客户：
```
[客户名称]您好，

非常抱歉，由于[原因]，项目可能需要延期[X]天。

原因说明：
[具体说明，不要找借口，客观陈述]

解决方案：
- 新的交付时间：[日期]
- 为保证质量，我们需要[具体措施]
- 作为补偿，我们可以[提供额外价值]

希望获得您的理解。如有疑虑，欢迎随时沟通。
```

**Step 2：协商解决**
- 说明真实情况
- 提供补救方案
- 适当补偿
- 重新约定交付时间

**Step 3：执行补救**
- 加快进度
- 保证质量
- 履行补偿承诺

## 违约预防策略

### 预防客户违约

**1. 合同条款明确**
- 付款时间具体到日期
- 明确逾期付款的后果
- 约定项目取消的费用

**2. 过程管理**
- 定期提醒付款节点
- 关键节点书面确认
- 保留所有沟通记录

**3. 客户筛选**
- 了解客户信誉
- 要求预付款
- 对高风险客户提高预付款比例

### 预防自己违约

**1. 合理承诺**
- 评估真实工作量
- 预留缓冲时间
- 不为了接项目而过度承诺

**2. 项目管理**
- 制定详细计划
- 定期检查进度
- 及早发现问题

**3. 及时沟通**
- 遇到困难立即告知
- 不要拖到最后一刻
- 提供解决方案而非只是问题

## 争议解决机制

### 友好协商
**优先选择**：成本低，维护关系
```
双方应本着友好合作的态度，通过协商解决争议。
协商应在争议发生后15日内完成。
```

### 第三方调解
**适用**：协商不成，仍希望非诉讼解决
```
协商不成的，可申请[行业协会/调解机构]调解。
调解费用由双方各承担50%。
```

### 仲裁/诉讼
**最后手段**：成本高，关系破裂
```
调解不成的，任何一方可向[甲方所在地/乙方所在地/项目所在地]
人民法院提起诉讼。
```

## 实用话术

### 客户拖延付款时

**温和提醒**：
```
[客户名称]您好，

项目已于[日期]完成并交付。
根据合同，尾款[X]元应在交付后3日内支付。

目前款项尚未到账，请帮忙确认一下支付进度。
如有任何问题，请及时告知。

谢谢！
```

**强硬催促**：
```
[客户名称]您好，

关于项目尾款[X]元，已逾期[X]天。
我方已多次提醒，但至今未收到款项。

根据合同第[X]条，逾期付款将产生违约金[X]元/日。

请立即处理付款事宜，否则我方将：
1. 暂停所有后续服务
2. 按合同约定追讨违约金
3. 保留采取法律措施的权利

请务必重视，[3日内]解决此事。
```

### 自己需要延期时

```
[客户名称]您好，

非常抱歉地通知您，由于[客观原因]，
原定于[日期]的交付需要延期至[新日期]。

延期原因：
[如实说明，不找借口]

影响评估：
- 延期[X]天
- 不会影响最终质量
- [其他影响说明]

补救措施：
- 我们已采取[措施]加快进度
- 作为补偿，我们将[提供额外价值]
- 确保[新日期]准时交付

再次为此次延期致歉，感谢您的理解。
如有任何疑虑，请随时沟通。
```
