## 核心概念

### 为什么要从执行转型到顾问？

**执行层的困境**：
- 收入天花板明显（按时间/项目收费）
- 工作内容重复，成长有限
- 容易被替代（年轻设计师成本低）
- 身体消耗大（加班、改稿）

**顾问层的优势**：
- 收入上限更高（按价值/时间收费）
- 工作内容更有挑战性（解决复杂问题）
- 不可替代性强（需要经验和洞察力）
- 工作方式更灵活（咨询、培训、顾问）

### 执行 vs 顾问的区别

| 维度 | 执行层 | 顾问层 |
|:---|:---|:---|
| **核心能力** | 设计技能 | 商业洞察 + 设计思维 |
| **服务对象** | 客户的"手" | 客户的"脑" |
| **交付物** | 设计稿、方案 | 策略、建议、解决方案 |
| **收费方式** | 按项目/时间 | 按价值/时间/ retainer |
| **客户关系** | 乙方-甲方 | 合作伙伴 |
| **工作方式** | 接单-执行-交付 | 诊断-策略-指导 |

## 转型的三个阶段

### 阶段1：执行+（1-2年）

**目标**：在执行中增加策略思维

**行动**：
1. **不只问"做什么"，多问"为什么"**
   - 客户为什么要做这个项目？
   - 业务目标是什么？
   - 如何衡量成功？

2. **提供额外价值**
   - 不只是执行brief，还提供建议
   - 指出brief中的问题
   - 提出更好的解决方案

3. **学习商业知识**
   - 了解客户所在行业
   - 学习基础商业概念（ROI、LTV、CAC等）
   - 关注商业案例和趋势

**示例**：
```
客户说："我们需要一个新的logo。"

执行层回应："好的，您想要什么风格？"

执行+回应："好的。在开始前，我想了解一下：
- 为什么要换logo？是品牌升级还是业务转型？
- 目标用户是谁？他们的审美偏好是什么？
- 新logo需要传达什么品牌信息？
- 预算多少？时间要求？

这样我能提供更符合您业务目标的方案。"
```

---

### 阶段2：策略执行（2-4年）

**目标**：成为"策略+执行"的复合型人才

**行动**：
1. **提供策略服务**
   - 品牌策略咨询
   - 视觉策略规划
   - 设计体系搭建

2. **建立方法论**
   - 形成自己的工作方法
   - 能够系统化地解决问题
   - 可以培训/指导他人

3. **提升影响力**
   - 分享专业观点
   - 建立个人品牌
   - 成为细分领域专家

**服务升级**：
```
从："我帮你设计logo"
到："我帮你梳理品牌策略，然后设计logo"

从："按项目收费5万"
到："策略咨询3万 + 设计执行5万 = 8万"
```

---

### 阶段3：纯顾问（4年+）

**目标**：成为独立顾问，不卖时间卖智慧

**行动**：
1. **纯咨询服务**
   - 不参与具体执行
   - 提供诊断、建议、方案
   - 指导客户团队执行

2. **多元化收入**
   - 咨询费（按小时/天）
   - Retainer（月度顾问费）
   - 培训/工作坊
   - 股权投资（用服务换股权）

3. **建立生态**
   - 培养执行团队/合作伙伴
   - 建立行业影响力
   - 成为行业专家/意见领袖

**收费模式**：
```
- 单次咨询：5000-20000元/天
- 月度顾问：20000-50000元/月
- 项目咨询：项目总额的10-20%
- 培训：10000-50000元/场
```

## 能力升级路径

### 硬技能升级

**1. 商业思维**
- 理解商业模式
- 会看财务报表
- 懂市场营销
- 了解用户心理

**2. 策略能力**
- 品牌策略
- 产品策略
- 增长策略
- 竞争策略

**3. 咨询技能**
- 诊断问题
- 引导讨论
- 呈现方案
- 推动落地

### 软技能升级

**1. 沟通能力**
- 与高管对话
- 引导客户思考
- 处理复杂关系
- 演讲和呈现

**2. 思维能力**
- 系统思考
- 批判性思维
- 创新思维
- 战略思维

**3. 影响力**
- 建立信任
- 说服他人
- 领导团队
- 行业影响

## 转型实操指南

### 步骤1：定位你的专业领域

**选择标准**：
- 你有丰富经验
- 市场需求大
- 竞争不激烈
- 你有兴趣深耕

**示例领域**：
- 新消费品牌视觉策略
- SaaS产品设计体系
- 传统企业数字化转型
- 创业公司品牌从0到1

### 步骤2：打造专业形象

**必备材料**：
- 专业简介（突出策略能力）
- 案例集（展示策略+执行）
- 方法论文章
- 客户证言

**渠道建设**：
- 公众号/知乎（深度内容）
- 即刻/小红书（专业形象）
- 行业活动（演讲/分享）
- 播客/视频（多媒体）

### 步骤3：获取第一个咨询客户

**策略**：
1. **从现有客户转化**
   - 对老客户说："除了设计，我还可以提供策略咨询"
   - 先低价或免费试一次
   - 证明价值后收费

2. **通过内容吸引**
   - 写策略相关的文章
   - 分享案例复盘
   - 吸引有需求的客户

3. **通过人脉推荐**
   - 告诉朋友你提供咨询服务
   - 参加行业活动认识潜在客户
   - 寻找合作伙伴推荐

**话术**：
```
"除了设计执行，我还提供品牌策略咨询服务。

我可以帮助您：
- 梳理品牌定位和核心价值
- 制定视觉策略和体系
- 指导团队执行

如果您有这方面的需求，我们可以聊聊。
首次咨询可以免费，看看能否帮到您。"
```

### 步骤4：设计咨询服务产品

**产品1：品牌诊断**
- 时间：1-2天
- 价格：5000-20000元
- 交付：诊断报告 + 改进建议

**产品2：策略咨询**
- 时间：按天或按项目
- 价格：5000-20000元/天
- 交付：策略方案 + 执行指导

**产品3：月度顾问**
- 时间：按月
- 价格：20000-50000元/月
- 交付：每月固定咨询时间 + 随时答疑

**产品4：培训工作坊**
- 时间：半天-2天
- 价格：10000-50000元/场
- 交付：定制化培训

## 常见挑战与应对

### 挑战1：客户不认你的顾问身份
**症状**：客户只把你当设计师
**应对**：
- 从设计项目切入，逐步展示策略能力
- 写专业文章建立权威
- 找愿意尝试的客户

### 挑战2：不知道怎么定价
**症状**：不敢收高价
**应对**：
- 参考同行价格
- 从低价开始，逐步提升
- 按价值定价，而非时间

### 挑战3：缺乏商业知识
**症状**：不懂客户的商业问题
**应对**：
- 系统学习商业知识（MBA课程、商业书籍）
- 多和客户聊业务
- 请商业顾问朋友指导

### 挑战4：收入不稳定
**症状**：咨询项目不持续
**应对**：
- 设计 retainer 产品，保证基础收入
- 同时保留部分执行项目
- 建立客户 pipeline
