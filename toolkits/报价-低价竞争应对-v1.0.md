# 应对低价竞争报价模组 v1.0

## 元信息
- **来源病例**：#007（客户比价压价，如何用价值差异化赢得项目）
- **共识形成日期**：2026-02-02
- **最后更新**：2026-02-02
- **贡献者**：Creative-Career-OS 社区
- **适用场景**：客户拿着其他低价报价来压价

## 核心策略
通过**价值差异化展示**、**服务分层报价**、**拒绝低价话术**，避免陷入价格战，用专业价值赢得合理报价。

## 详细内容

### 方案A：价值锚定法

**核心思路**：不直接回应价格，而是重新定义价值标准，让客户理解"贵有贵的道理"。

**话术模板**：

**场景：客户说"另一家报价只有你们的一半"**

```
感谢您的坦诚。关于价格差异，我想和您分享一下我们的价值所在：

**成本结构对比：**

| 成本项 | 低价方案 | 我们的方案 |
|:---|:---|:---|
| 设计执行 | ✓ | ✓ |
| 需求调研 | ✗ | ✓（2-3天深度调研） |
| 策略咨询 | ✗ | ✓（品牌定位建议） |
| 修改次数 | 1-2轮 | 3轮+ |
| 源文件交付 | 部分 | 完整 |
| 售后支持 | 无 | 3个月内免费咨询 |
| 案例经验 | 新手 | 50+同类项目经验 |

**风险对比：**
- 低价方案风险：设计不符合定位、反复修改超预算、最终效果不达预期
- 我们的保障：前期充分调研降低返工风险、经验丰富减少试错成本

**实际成本计算：**
- 低价方案：报价[X]元 + 返工成本[X]元 + 时间成本[X]元 = 实际[X]元
- 我们的方案：报价[X]元，一次到位

您看，从总成本角度，哪个方案更划算？
```

### 方案B：服务分层法

**核心思路**：提供不同档次的服务包，让客户自己选择，既保留客户又维护价格体系。

**话术模板**：
```
理解您的预算考虑。为了满足不同需求，我准备了三个方案：

**方案A：基础版（[X]元）**
- 适用场景：预算有限，需求明确
- 包含内容：
  - 1个设计方向
  - 1轮修改
  - 基础格式交付物
- 不包含：策略咨询、深度调研、源文件

**方案B：标准版（[X]元）- 推荐**
- 适用场景：大多数项目
- 包含内容：
  - 2-3个设计方向
  - 3轮修改
  - 完整交付物+源文件
  - 基础策略建议
- 附加价值：降低返工风险，提高项目成功率

**方案C：完整版（[X]元）**
- 适用场景：战略级项目，需要深度服务
- 包含内容：
  - 全套方案B内容
  - 深度市场调研
  - 品牌策略咨询
  - 6个月内持续支持
- 附加价值：不仅是设计，更是品牌升级

您看哪个方案更符合您的需求？或者我们可以针对您的具体情况定制？
```

### 方案C：拒绝低价法

**核心思路**：当客户坚持低价且无法协商时，礼貌但坚定地拒绝，保护自身价值。

**话术模板**：

**初级拒绝**（温和）：
```
感谢您的考虑。关于价格，我们已经是最诚实的报价了。

我们的定价基于：
1. 实际工作量和时间成本
2. 专业经验和案例积累
3. 对项目质量的承诺

如果预算确实有限，我建议：
- 可以先选择基础版方案
- 或者等项目预算充足后再合作
- 我也可以推荐其他适合您预算的设计师

希望未来有机会合作！
```

**进阶拒绝**（专业）：
```
理解您的预算压力。但经过评估，这个价格我们无法保证项目质量。

**原因：**
1. 低于成本价：这个报价甚至覆盖不了我们的基础成本
2. 质量风险：为了压缩成本，只能减少必要的工作环节
3. 长期影响：低质量交付对双方声誉都不好

**建议：**
- 如果预算确实紧张，建议先简化项目范围
- 或者分阶段执行，先做最核心的部分
- 我可以帮您规划一个符合预算的简化方案

如果以上方案都无法接受，我只能遗憾地放弃这个项目。但我可以推荐其他可能适合的设计师。

祝项目顺利！
```

## 使用指南

### 何时使用
- ✅ 客户拿着其他低价报价来比价
- ✅ 客户说"预算有限，能不能便宜点"
- ✅ 客户暗示"价格太高，我们要考虑其他家"
- ✅ 客户直接要求降价

### 何时不适用
- ❌ 长期合作老客户的小调整
- ❌ 客户确实有合理预算限制（应调整方案而非降价）
- ❌ 项目本身价值确实不高（应直接拒绝而非降价）

### 注意事项
1. **不要直接降价**：一旦降价，客户会认为你的报价有水分
2. **强调价值而非价格**：让客户理解价格背后的价值
3. **提供选择**：分层服务给客户决策权
4. **准备退出**：不是所有项目都值得接，学会说"不"
5. **保持尊重**：即使拒绝，也要保持专业和礼貌

## 识别"价格敏感型客户"信号

以下信号表明客户可能只关注价格，不关注价值：
- ⚠️ 第一句话就问"多少钱"
- ⚠️ 对设计流程和专业性没有兴趣
- ⚠️ 拿着多家报价单纯比价
- ⚠️ 说"别家便宜很多"但不提别家的方案
- ⚠️ 对质量保障和售后服务不关心

**建议**：这类客户往往后期事多、付款慢，谨慎接单。
