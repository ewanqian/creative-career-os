# 应对无预算试探报价模组 v1.0

## 元信息
- **来源病例**：#006（从客户拒绝透露预算到成功报价8万）
- **共识形成日期**：2026-02-02
- **最后更新**：2026-02-02
- **贡献者**：Creative-Career-OS 社区
- **适用场景**：客户询价时不愿透露预算或声称"没有预算"

## 核心策略
通过**预算探测话术**、**价值锚定**、**区间报价法**等技巧，在不吓跑客户的前提下，获取预算信息并掌握报价主动权。

## 详细内容

### 方案A：三级探测法

**核心思路**：通过三个层级的提问，逐步引导客户透露预算范围。

**话术模板**：

**第一级：范围探测**
```
"理解您的需求了。这类项目的投入跨度比较大，从[X]万到[X]万都有可能，
主要取决于[具体因素，如：创意复杂度、应用场景数量、时间紧迫程度等]。

您这边的预算范围大概是怎样的？这样我可以给您推荐最适合的方案。"
```

**第二级：方案对比**
```
"根据您的需求，我准备了三个不同深度的方案供参考：

方案A（基础版）：[X]元，包含[核心内容]，适合[适用场景]
方案B（标准版）：[X]元，包含[扩展内容]，适合[适用场景]  
方案C（完整版）：[X]元，包含[全套内容]，适合[适用场景]

您看哪个方案比较符合您的预期？"
```

**第三级：细节确认**
```
"好的，那我们聚焦在方案[X]。为了确保最终效果，我想再确认几个细节：
[具体问题1]
[具体问题2]
...

基于这些，最终报价可能在[X-X]元之间。您觉得这个范围可以接受吗？"
```

### 方案B：价值先行法

**核心思路**：先展示价值，建立价格锚点，再让客户自己评估预算匹配度。

**话术模板**：
```
"在谈价格之前，我想先和您分享一下这个项目能为您带来的价值：

1. [价值点1：如提升品牌辨识度，预计带来XX效果]
2. [价值点2：如优化用户体验，预计提升转化率XX%]
3. [价值点3：如建立视觉体系，降低未来设计成本]

基于这些价值，我们的专业投入需要[X]元。

当然，如果预算有限，我们也可以先聚焦最核心的[某部分]，费用为[X]元，
后续再根据效果决定是否追加投入。

您更倾向于哪种方式？"
```

### 方案C：反向提问法

**核心思路**：当客户说"没有预算"或"你说多少就多少"时，通过专业提问夺回主动权。

**话术模板**：
```
"明白，很多项目在初期确实还没有确定预算。为了给您一个准确的报价，
我需要了解几个关键点：

1. 这个项目的时间节点是怎样的？是否有硬性deadline？
2. 项目决策流程是怎样的？需要经过哪些人的确认？
3. 类似的项目您之前是否有做过？当时的投入大概是多少？
4. 这个项目对业务的重要性如何？是战略级还是支持级？

了解这些后，我可以给出一个既符合您需求又在合理预算范围内的方案。
您看我们什么时候可以详细沟通一下？"
```

## 使用指南

### 何时使用
- ✅ 客户初次询价，不愿透露预算
- ✅ 客户说"没有预算，你先报个价"
- ✅ 客户预算明显低于项目实际成本
- ✅ 需要快速筛选"优质客户"和"价格敏感客户"

### 何时不适用
- ❌ 已有长期合作基础，双方信任度高
- ❌ 公开招标项目，预算已明确公示
- ❌ 客户已明确表示"价格不是问题"

### 注意事项
1. **不要先报价**：先了解需求和价值，再谈价格
2. **给出区间**：避免一个固定数字，给客户决策空间
3. **强调价值**：让客户理解价格背后的价值，而非单纯比较数字
4. **设置底线**：明确自己的最低接受价格，不轻易妥协
5. **准备退出**：如果预算实在无法匹配，礼貌地推荐其他解决方案

## 报价策略

### 区间报价技巧
```
"根据您的需求，这类项目的投入通常在[X]万到[X]万之间。
具体取决于以下几个因素：
- [因素1]：[选项A]或[选项B]，差价约[X]元
- [因素2]：[选项A]或[选项B]，差价约[X]元
- [因素3]：[选项A]或[选项B]，差价约[X]元

您可以根据实际情况选择最适合的组合。"
```

### 锚定效应运用
```
"像[某知名品牌]类似的案子，他们投入了[X]万，最终效果[具体成果]。

当然，每个项目情况不同。基于您的具体需求，我建议的投入是[X]万，
这样可以确保[核心成果]，同时控制在合理预算内。"
```
