# 沟通-客户拖延付款

## 元信息
- **来源真实故事**：#018（项目交付后客户拖延付款的应对）
- **共识形成日期**：2026-02-02
- **最后更新**：2026-02-02
- **贡献者**：Creative-Career-OS 社区
- **适用场景**：项目已完成但客户迟迟不付款

## 核心策略
通过**分级催款**、**法律威慑**、**关系维护**三重手段，既坚决维护自身权益，又尽量保持客户关系。

---

## 拖延付款的常见原因

### 客户方原因
- **资金紧张**：公司现金流问题
- **审批流程**：大公司内部流程慢
- **不满意成果**：对交付物有意见但不直说
- **恶意拖欠**：故意拖延试探底线
- **对接人变更**：原对接人离职或调岗

### 自身原因
- 合同付款条款不清晰
- 未按约定节点收款
- 交付物未获正式确认
- 缺乏催款意识和流程

## 分级催款策略

### 第一阶段：友好提醒（逾期1-7天）

**目标**：善意提醒，给客户台阶

**沟通方式**：微信/邮件

**话术模板**：
```
[客户名称]您好，

[项目名称]已于[日期]完成交付，非常感谢您的信任！

根据合同约定，尾款[X]元应在交付后[3日]内支付。
想确认一下付款进度，是否遇到什么流程问题？

如有任何需要配合的地方，请随时告知。

谢谢！
[你的名字]
```

**关键要点**：
- 语气友好，不指责
- 提醒合同依据
- 给对方台阶（"流程问题"）
- 表达配合意愿

---

### 第二阶段：正式催促（逾期8-15天）

**目标**：明确严肃，表明态度

**沟通方式**：邮件+微信

**话术模板**：
```
主题：[项目名称]尾款支付提醒 - 逾期[X]天

[客户名称]您好，

关于[项目名称]的尾款[X]元，已逾期[X]天。

我方已于[日期]发送提醒，但至今未收到款项。

根据合同第[X]条约定：
"甲方应在项目验收后3日内支付尾款，逾期按日0.5%支付违约金。"

请尽快安排付款，以免：
1. 产生额外违约金（目前累计[X]元）
2. 影响后续合作
3. 需要启动正式催款流程

请[3个工作日内]完成支付，或与我联系说明情况。

附件：
- 合同扫描件
- 交付确认书
- 发票

[你的名字]
[联系方式]
```

**关键要点**：
- 引用合同条款
- 明确逾期后果
- 给出具体期限
- 保留证据

---

### 第三阶段：严肃警告（逾期16-30天）

**目标**：施加压力，准备法律手段

**沟通方式**：正式邮件+电话

**话术模板**：
```
主题：[公司名称]关于[项目名称]尾款支付的正式催款函

致：[客户公司名称]

关于贵我双方于[日期]签订的《[项目名称]合同》，
我方已按约定完成全部工作并于[日期]交付，
贵方已于[日期]签署《项目验收确认书》。

根据合同第[X]条，贵方应于[日期]前支付尾款[X]元。
截至目前，已逾期[X]天，累计违约金[X]元。

我方已多次催款（详见附件沟通记录），
但贵方至今未履行付款义务。

现正式要求贵方：
1. 于收到本函后[5个工作日]内支付全部欠款[X]元
2. 支付逾期违约金[X]元
3. 如确有困难，请提供书面还款计划

如贵方在上述期限内仍未付款，
我方将不得不采取以下措施：
1. 暂停所有正在进行中的项目
2. 委托律师发送律师函
3. 通过法律途径追讨欠款
4. 在行业信用平台公示违约行为

希望贵方重视此事，尽快解决。

特此函告。

[你的名字]
[日期]

附件：
1. 合同
2. 交付确认书
3. 发票
4. 过往沟通记录
```

**关键要点**：
- 正式函件格式
- 列举所有证据
- 明确法律后果
- 保留最后协商空间

---

### 第四阶段：法律手段（逾期30天以上）

**目标**：通过法律途径追讨

**步骤1：律师函**
- 委托律师发送正式律师函
- 费用：约500-2000元
- 威慑力强，很多客户到此阶段会付款

**步骤2：调解/仲裁**
- 如合同有仲裁条款，申请仲裁
- 或申请行业协会调解
- 费用较低，时间较快

**步骤3：诉讼**
- 向法院提起诉讼
- 准备材料：合同、交付证据、沟通记录、发票
- 时间：3-6个月
- 费用：诉讼费+律师费（胜诉后可要求对方承担）

**步骤4：执行**
- 胜诉后申请强制执行
- 法院查封对方账户/财产

## 不同场景应对策略

### 场景1：客户说"财务流程慢"

**应对**：
```
"理解大公司的流程。能否提供一个预计付款时间？
同时，能否先支付一部分（如50%）以表示诚意？
我们可以签署一个付款承诺书，明确剩余款项的支付日期。"
```

**行动**：
- 要求书面付款承诺
- 要求部分付款
- 明确最终期限

---

### 场景2：客户说"老板不满意"

**应对**：
```
"非常抱歉听到这个消息。能否具体说明哪些方面不满意？
我们在[日期]交付时，[对接人]已签署验收确认书。

如果确实存在问题，我们愿意：
- 在合理范围内进行修改
- 协商解决方案

但请注意，这不能成为拖延付款的理由。
根据合同，验收后应按时付款，后续修改是另一回事。"
```

**行动**：
- 要求具体反馈
- 区分"不满意"和"拖延付款"
- 愿意合理修改，但坚持付款
- 如对方无理取闹，进入法律程序

---

### 场景3：客户对接人离职

**应对**：
```
"得知[前对接人]离职，理解交接需要时间。
请新的对接人联系我，我会重新说明项目情况和付款事宜。

同时，请确认：
1. 项目已交付并验收（有验收确认书）
2. 付款流程是否正常进行
3. 是否需要我提供任何补充材料

希望能尽快完成交接，不影响付款进度。"
```

**行动**：
- 主动联系新对接人
- 提供完整项目资料
- 重新确认付款流程
- 适当给予交接时间（1-2周）

---

### 场景4：客户恶意拖欠

**识别信号**：
- 多次承诺但不兑现
- 找各种借口推脱
- 态度恶劣或失联
- 公司出现经营问题传闻

**应对策略**：
1. **立即停止所有工作**
   - 不交付源文件
   - 不进行任何修改
   - 已交付的考虑撤回使用权

2. **全面证据保全**
   - 合同、验收书、发票
   - 所有沟通记录截图
   - 对方公司工商信息

3. **快速法律行动**
   - 律师函
   - 财产保全（如金额大）
   - 诉讼

4. **行业公示**
   - 在同行群提醒
   - 在行业平台标记
   - （注意脱敏，避免法律风险）

## 预防措施

### 合同层面
- 明确付款节点和逾期违约金
- 约定"交付即视为验收，3日内无异议默认验收"
- 保留源文件直至尾款付清

### 流程层面
- 每个付款节点提前3天提醒
- 交付时同步发送invoice
- 获得书面验收确认后再交付源文件

### 客户筛选
- 新客户要求更高比例的预付款
- 了解客户信誉（同行打听）
- 对高风险客户要求分期付款

## 心态建设

### 催款不是乞讨
- 这是你应得的劳动报酬
- 客户违约，不是你的错
- 坚决维护权益是专业表现

### 不要害怕失去客户
- 拖欠款项的客户不是好客户
- 及时止损，避免更大损失
- 好客户会理解并按时付款

### 法律是最后手段，但也是有力手段
- 大多数客户不想惹上官司
- 律师函威慑力很强
- 诉讼虽然麻烦，但能追回欠款
